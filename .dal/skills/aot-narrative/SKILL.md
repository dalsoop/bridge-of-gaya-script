# Skill: AoT Narrative

진격의거인식 내러티브 기법을 가야의 연결점에 적용한다.

> 출처: 이사야마 하지메의 진격의 거인 내러티브 분석 기반

## 기법 1: 이중 레이어 대사 (Dual-Layer Dialogue)

**모든 중요한 대사는 표면 의미 + 숨겨진 의미를 동시에 가져야 한다.**
표면만으로도 자연스럽게 읽혀야 하며, 숨겨진 의미는 나중에야 활성화.

### 적용 맵

| 대사 | 표면 의미 | 숨겨진 의미 | 회수 시점 |
|------|-----------|-------------|-----------|
| "…또 왔네." | 데자뷔? | 수백 번째 만남의 기억 | 22화 |
| "…고마워." (프롤로그) | 왜 고마워? | 루프에서 해방시켜줘서 | 25화 |
| "이런 거 좋아하는 거 어떻게 알았어?" | 우연히 맞춤? | 수백 루프의 데이터로 이미 알고 있음 | 22화 |
| "같이 살았으면 좋았을 텐데." | 소박한 바람 | 루프 밖의 삶을 꿈꿈 | 25화 |
| "이번엔 좀 빨랐다." | 빨리 돌아왔네 | 이전 루프보다 사냥이 빨랐다는 비교 | 22화 |
| "계속 될 리 없지만." | 비관적 성격 | 5일 뒤 루프 리셋을 알고 있음 | 22화 |
| "…무서워." | 겁쟁이 | 매일 밤 죽는 기억 때문 | 19화 |
| "…괜찮아." | 별일 아님 | 상처를 숨기는 거짓말 | 12화 |
| "죽여야 다시 태어날 수 있어." | 미쳐서 하는 말 | 리스폰 메커니즘의 직접 설명 | 25화 |
| 칼리스 "나도 해봤거든." | 공감 표현 | 이전 루프의 자기가 했던 일 | 24화 |

### 작성 규칙
```
1. 대사를 쓸 때 [표면]과 [심층]을 동시에 정의
2. [표면]만으로 대화가 완전히 성립해야 함
3. [심층]은 특정 화에서만 활성화
4. 플레이어가 2회차에서 같은 대사를 보면 "아!"가 나와야 함
```

## 기법 2: 선택적 내면 은폐 (Selective POV Withholding)

**비밀이 있는 캐릭터의 내면은 보여주지 않는다. 플레이어가 눈치채면 안 됨.**

### 적용

| 캐릭터 | 내면 접근 | 이유 |
|--------|-----------|------|
| 테이 | ✅ 접근 가능 | 플레이어 = 테이 |
| 가야 (페르소나) | ❌ 접근 불가 | 마스킹. 진짜 생각을 모름 |
| 가야 (본체, 23화~) | ✅ 제한적 접근 | 마스킹 해제 후 |
| 칼리스 | ❌ 접근 불가 | 정체 비밀 |
| 기사단 NPC | ❌ 접근 불가 | 루프 인식 숨김 |

### VN 구현
- 1회차: 테이 시점만
- 2회차(New Game+): **가야 시점 내면 독백이 추가로 보임**
  - 같은 장면인데 가야의 내면이 새로 뜸
  - 예: 1화에서 "…또 왔네." → 2회차에서 가야 내면: "(…847번째. 이번에는 어디까지 알고 올까.)"

## 기법 3: 반클라이맥스 공개 (Anticlimactic Reveal)

**가장 큰 반전은 조용하게 던진다. 극적 연출이 아니라 일상 대화 속에 묻는다.**

### 진격의거인 예시
라이너가 "나는 갑옷 거인이야"를 배경 대화에 묻어서 말함.
드라마틱 연출 없음. 오히려 다른 대화에 묻힘. → 더 충격적.

### 우리 프로젝트 적용
- **22화**: 가야가 "또 왔네"의 진실을 말하는 장면
  - ❌ 극적 CG + 음악 폭발
  - ✅ 평범한 대화 흐름 중에 담담하게:

가야: "오늘 뭐 할래?"

테이: (선택지)

가야: "…아, 참. '또 왔네' 있잖아."

가야: "데자뷔 아니야."

가야: "기억이야."

가야: "너를 기억해. 수백 번."

가야: "…그래서 오늘 뭐 할래?"

- **대화가 이어짐. 가야가 "당연한 듯" 말함. 그게 더 무서움.**

## 기법 4: 물건으로 관계 말하기 (Object-as-Relationship)

**물리 오브젝트 하나에 관계의 전체 감정을 싣는다.**

### 리본의 AoT식 활용

| 시점 | 리본의 상태 | 관계의 상태 |
|------|------------|-------------|
| 3화 | 새 리본, 선물 | 시작, 설렘 |
| 7화 | 피 묻은 리본, 드랍 | 균열, 충격 |
| 루프 리셋 | 리본 사라짐 | 관계도 리셋? |
| 2회차 | 또 같은 리본 선물 | 반복 |
| 24화 | 가야가 "매번 줘. 매번 흘려." | 사랑과 죽음이 같은 오브젝트 |
| 루트A | 가야 소멸 후 리본만 남음 | 가야가 떠난 뒤 남는 유일한 것 |

### 추가: 루트A 엔딩에 리본 씬 추가 필요
가야가 빛으로 흩어진 뒤, 바닥에 **리본만 남아있는** 컷.
테이가 리본을 집어드는 CG. → **E1 엔딩 CG를 이걸로 교체.**

## 기법 5: 모순이 깊이다 (Contradiction as Depth)

**캐릭터가 모순적일수록 깊어진다. 한 가지로 규정 가능하면 얕다.**

### 가야의 3중 자아 (라이너 참조)

| 자아 | 정의 | AoT 대응 |
|------|------|----------|
| **본래 가야** | 루프 이전의 행복한 소녀 | 본래 라이너 |
| **루프 가야** | 죽음을 반복하며 망가진 존재 | 전사 라이너 |
| **연기 가야** | 플레이어 앞에서 "괜찮은 척" | 병사 라이너 |

**핵심**: 연기 가야의 다정함은 가짜가 아님. 진심으로 좋아함. 하지만 동시에 루프 가야의 공포도 진짜. 플레이어는 "어디까지가 진짜야?"를 결정할 수 없어야 함.

### 칼리스의 모순
- 진실을 추구하면서도 자기 정체를 모름
- 플레이어를 돕지만, 도움의 동기가 "자기가 실패한 걸 대신 해달라"는 이기심
- 냉철하면서도 소멸 직전에 "고마워"

## 기법 6: 진짜 배신 (Genuine Betrayal)

**배신자의 애정이 진짜여야 배신이 아프다.**

### 우리 프로젝트에서
가야는 배신자가 아니지만, **"가야가 처음부터 알고 있었다"는 22화 반전**이 같은 구조.

❌ 나쁜 구현: "가야가 플레이어를 조종하고 있었다"
✅ 좋은 구현: "가야가 알면서도 진심으로 행복했다. 알고 있는 것과 느끼는 것은 별개."

가야 22화 추가 대사:

가야: "다 알고 있었어."

가야: "근데 그래도… 오늘 같은 날은 진짜로 좋았어."

가야: "알면서 좋아하는 게 더 아프다는 거… 알아?"

## 기법 7: New Game+ 재맥락화

**2회차 플레이에서 이전에 보이지 않던 텍스트가 추가됨.**

### 구현 방안

| 조건 | 추가되는 것 | 예시 |
|------|-------------|------|
| 엔딩 1회 클리어 | 가야 내면 독백 | 1화 "…또 왔네." + "(847번째. 이번엔 뭐가 다를까.)" |
| 루트A 클리어 | 칼리스 내면 독백 | 10화 "너 또 바뀌었네." + "(나도 저랬겠지. 그때의 나도.)" |
| 루트C 클리어 | 프롤로그 확장 | 프롤로그에 가야 시점 추가: "(드디어 끝낼 수 있어. 고마워.)" |
| 전 루트 클리어 | 히든 에필로그 | 가야의 편지 형태로 전체 루프를 회고 |

## 기법 8: 한 줄이 3가지를 하게 (Every Line Does 3 Things)

**모든 대사가 최소 2가지 기능을 동시에 수행해야 한다.**

| 기능 | 설명 |
|------|------|
| 캐릭터 노출 | 이 대사로 캐릭터의 성격/상태를 보여주는가 |
| 플롯 진행 | 이 대사가 이야기를 앞으로 밀어주는가 |
| 분위기 | 이 대사가 씬의 톤을 만드는가 |
| 떡밥 기능 | 이 대사가 나중에 다른 의미로 돌아오는가 |

### 대사 검증 체크리스트
```
대사: "…이번에는… 괜찮을까…"

캐릭터: ✅ 가야의 불안함
플롯:   ✅ 밤의 공포 → 다음 날로 전진
분위기: ✅ 불안한 밤의 마무리
떡밥:   ✅ "이번에는" = 루프 인식 (22화 회수)

기능 수: 4개 → 우수
```

```
대사: "오늘 날씨 좋다."

캐릭터: ❌ 누구나 할 수 있는 말
플롯:   ❌ 이야기 안 움직임
분위기: △ 평온
떡밥:   ❌ 없음

기능 수: 0.5개 → 삭제 또는 교체
```

## 에피소드 적용 체크리스트

에피소드 작성/검토 시 아래를 확인:

- [ ] 핵심 대사에 이중 레이어가 있는가 (기법 1)
- [ ] 비밀 캐릭터의 내면이 노출되지 않았는가 (기법 2)
- [ ] 가장 큰 반전이 조용하게 전달되는가 (기법 3)
- [ ] 리본(또는 핵심 오브젝트)이 씬에서 관계를 대변하는가 (기법 4)
- [ ] 가야가 한 가지로 규정되지 않는가 (기법 5)
- [ ] "알면서 좋아하는" 구조가 유지되는가 (기법 6)
- [ ] 2회차 추가 텍스트가 준비되어 있는가 (기법 7)
- [ ] 모든 대사가 최소 2가지 기능을 하는가 (기법 8)
