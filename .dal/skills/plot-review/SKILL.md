# Skill: Plot Review

에피소드 또는 전체 플롯을 검토할 때 아래 체크리스트를 따른다.

## 체크리스트

### 1. 모순 검사
- [ ] 캐릭터 설정과 행동이 일치하는가
- [ ] 시간순서(루프/일차)가 맞는가
- [ ] 이전 화의 결과가 다음 화에 반영되는가
- [ ] 뒷배경(이든 등)이 게임 플롯에 직접 노출되지 않는가

### 2. 떡밥 검사
- [ ] 모든 Plant에 대응하는 Payoff가 있는가
- [ ] Payoff가 Plant보다 먼저 나오지 않는가
- [ ] 떡밥 레이어가 최소 2단계 이상인가 (같은 떡밥이 2번 이상 의미 변환)
- [ ] 회수되지 않는 떡밥이 없는가

### 3. 감정 곡선
- [ ] 각 Phase마다 감정 고조점이 있는가
- [ ] 핵심 화(7, 12, 19, 24, 27화)의 충격이 이전 화보다 강한가
- [ ] 평탄한 구간이 3화 이상 연속되지 않는가

### 4. 캐릭터 일관성
- [ ] 가야의 말투가 Phase별로 변화하는가 (순함→불안→집착→본체)
- [ ] 칼리스의 대사가 "이전 테이" 정체성과 모순되지 않는가
- [ ] 기사단 NPC 대사가 떡밥 기능을 수행하는가

### 5. 엔딩 정합성
- [ ] 27화 분기 조건이 명확한가
- [ ] 각 루트의 조건이 겹치지 않는가
- [ ] 루트E(에이전트)의 봉인석 복선이 5화부터 깔려있는가
