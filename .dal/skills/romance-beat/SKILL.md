# Skill: Romance Beat

로맨스 진행 비트를 체크하고 친밀도 사다리/push-pull 리듬을 관리한다.

## 친밀도 사다리 (Intimacy Ladder)

관계는 아래 순서를 따라야 하며, **2단계 이상 건너뛰면 부자연스럽다.**

```
1. 타인      — 경계, 거리감
2. 인식      — 존재를 의식함
3. 호기심    — 알고 싶어짐
4. 편안함    — 같이 있는 게 자연스러움
5. 취약성 공유 — 약한 모습을 보여줌
6. 감정적 의존 — 없으면 불안
7. 신체적 친밀 — 스킨십, 물리적 가까움
8. 완전 신뢰  — 비밀/진실 공유
```

### 현재 프로젝트 매핑

| Phase | 루프 | 친밀도 단계 | 주요 비트 |
|-------|------|-------------|-----------|
| P1 (1~7화) | 1~2 | 1→3 | 만남, 호기심, 첫 데이트 |
| P2 (8~15화) | 2~3 | 3→5 | 편안함, 취약성(공포 노출) |
| P3 (16~22화) | 4~5 | 5→6 | 의존, 진실 접근 |
| P4 (23~27화) | 최종 | 6→8 | 완전 신뢰, 비밀 공유, 선택 |

## Push-Pull 리듬

로맨틱 텐션 = **가까워짐(push)과 멀어짐(pull)의 반복**.

### 규칙
- 가까워지는 씬 2~3개 후 **반드시 거리가 생기는 이벤트** 1개
- 거리 원인: 밤 사냥 출발 / 칼리스 개입 / 가야 공포 / 마스킹 글리치 / 진실 접근
- 거리 후 다시 가까워질 때 **이전보다 한 단계 더 깊어져야** 함

### Push-Pull 예시
```
3화: 데이트 + 리본 선물 (push - 가까움)
4화: 밤 사냥 출발, 가야 공포 (pull - 거리)
5화: 상처 발견, 걱정 (push - 더 깊은 가까움)
7화: 피 묻은 리본 (pull - 충격으로 거리)
```

## 17+ 로맨스 톤 가이드

### 허용
- 감정 묘사: 직접적, 깊이 있게
- 스킨십: 손잡기, 기대기, 안기, 머리 쓰다듬기
- 암시: 분위기, 시선, 호흡, 심장 박동 묘사
- fade-to-black: 친밀한 상황 암시 후 장면 전환
- morning-after: 다음 날 아침의 분위기로 암시

### 금지
- 노골적 성적 묘사
- 강제적 상황의 로맨틱화

### 핵심 원칙
**물리적 친밀감보다 감정적 친밀감이 선행해야 한다.**
스킨십은 "이벤트"가 아니라 "감정의 결과"여야 한다.

## 검증 체크리스트

- [ ] 친밀도 단계가 2단계 이상 건너뛴 곳 없는가
- [ ] Push 후 2~3씬 이내에 Pull이 있는가
- [ ] Pull 후 다음 Push가 이전보다 깊은가
- [ ] 스킨십 씬에 감정적 맥락이 충분한가
- [ ] Phase별 친밀도 진행이 매핑 테이블과 일치하는가
- [ ] 가야의 공포/집착이 로맨스와 자연스럽게 얽히는가

## 출력 형식

```markdown
## EP.XX 로맨스 비트

| 씬 | 유형 | 친밀도 단계 | push/pull |
|----|------|-------------|-----------|
| S3 | 데이트 | 3→4 | push |
| S5 | 밤 공포 | 4(유지) | pull |
| S6 | 위로 | 4→5 | push (deeper) |

판정: ✅ push-pull 리듬 양호, 단계 건너뜀 없음
```
