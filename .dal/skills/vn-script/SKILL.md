# Skill: VN Script

비주얼노벨/어드벤처 게임 스크립트 포맷 및 연출 규칙.

## 대사 포맷

```
캐릭터명 (표정): "대사 텍스트"
```

예:
```
가야 (미소): "오늘도 같이 있어줄 거지?"
가야 (불안): "…밤이 무서워."
테이 (내면): (왜 이렇게 나를 무서워하지?)
```

### 대사 길이 규칙
- **한 텍스트 박스당 2~3문장 이내**
- 긴 독백은 여러 박스로 분리
- 침묵(……)은 별도 박스 처리

## 씬 헤더 포맷

```yaml
scene_id: ep01_s3
location: 거점 주변
time: 점심
bgm: daily_calm_01
background: bg_base_daytime
characters: [가야(기본), 테이]
mode: ADV
```

## 선택지 포맷

```markdown
**[선택지]**
1. "밖에 나가볼까?" → flags: [explore_day1] → 다음: ep01_s4a
2. "여기서 얘기하자." → flags: [stay_home_day1] → 다음: ep01_s4b
3. "혼자 둘 시간 좀 줘." → flags: [alone_day1, gaya_sad+1] → 다음: ep01_s4c
```

### 선택지 규칙
- 각 선택지는 **감정적 톤이 달라야** 한다 (같은 말 다른 표현 X)
- 숨겨진 태그(flags)를 반드시 명시
- 즉시 결과 + 장기 결과를 구분

## 연출 지시 포맷

```
[연출] 화면 0.5초 글리치
[연출] 가야 표정 전환: 미소 → 무표정 (0.3초)
[연출] BGM 페이드아웃 → 무음 2초 → BGM 변경: night_tension_01
[연출] 화면 암전. 텍스트: "DAY 1 END."
[CG] cg_ribbon_blood — 피 묻은 리본 클로즈업
[SFX] sfx_scream_distant — 먼 곳 비명
[SFX] sfx_glitch_short — 0.3초 노이즈
```

## 에셋 태깅

모든 씬에서 필요한 에셋을 명시:

```yaml
assets:
  sprites:
    - 가야_기본_미소
    - 가야_기본_불안
    - 가야_기본_무표정
  backgrounds:
    - bg_base_daytime
    - bg_base_night
  bgm:
    - daily_calm_01
    - night_tension_01
  sfx:
    - sfx_scream_distant
  cg:
    - cg_ribbon_blood
```

## 텍스트 연출 규칙

### 침묵/여운
```
가야: "……"
가야: "…괜찮아."
```
→ "……"과 대사를 **별도 박스**로. 침묵에 무게를 준다.

### 글리치 텍스트
```
가야: "나 너 ██해."
[연출] 0.3초 노이즈. 텍스트 깨짐.
가야: "…어? 내가 뭐라고 했지?"
```

### 선택지 붕괴 (26화)
```
**[선택지]**
1. "…"
2. "…"
3. "…"
→ 내부 분기는 호감도/이력 기반. 플레이어 선택과 무관.
```

### 강제 진행
```
[시스템] 선택지 없음. 강제 진행.
```
→ 1화, 밤 사냥 강제 등에서 사용. 플레이어 통제 박탈 연출.

## 화면 모드

| 모드 | 용도 |
|------|------|
| ADV | 일반 대화 (하단 텍스트박스) |
| NVL | 긴 내면 독백, 기억 재생 (전체화면 텍스트) |
| CG | 일러스트 전체화면 표시 |
| SYSTEM | 메타 연출 (루프 카운터, OS 알림 등) |

## 검증 체크리스트

- [ ] 모든 대사가 2~3문장 이내인가
- [ ] 모든 씬에 에셋 태깅이 되어있는가
- [ ] 선택지에 flags가 명시되어있는가
- [ ] 연출 지시가 구체적인가 (시간, 효과, 대상 명시)
- [ ] 침묵 박스가 대사와 분리되어있는가
- [ ] 글리치 연출에 복구 대사가 따라오는가
