# 30화 서사 전개 로드맵

> 이든/왕의 과거사는 뒷배경. 게임 안에서는 테이-가야-루프-추리만 다룬다.
> 기사단 동료들은 기억 공간 속 NPC로 등장하며, 각자 위치에서 간접 힌트를 준다.

## 화수와 루프의 관계

30화는 "에피소드 넘버"이지, 게임 내 일수가 아니다.
게임은 5일 루프를 여러 번 반복하며, 화수는 서사적 비트(beat)에 대응한다.

| 루프 | 게임 일수 | 대응 화수 | 서사 위치 |
|------|-----------|-----------|-----------|
| 1회차 | Day 1~5 | 1~6화 | Phase 1: 첫인상 + 첫 밤 + 첫 루프 |
| 1→2 전환 | 리셋 | 6화 | 루프 발생 인지 |
| 2회차 | Day 1~5 | 7~11화 | Phase 1 후반 + Phase 2 초반 |
| 3회차 | Day 1~5 | 12~15화 | Phase 2 후반: 핵심 물증들 |
| 4회차 | Day 1~5 | 16~19화 | Phase 3: 구조 파악 |
| 5회차 | Day 1~5 | 20~22화 | Phase 3 후반: 칼리스/가야 진실 |
| 최종 | Day 1~5 | 23~27화 | Phase 4: 본체 대면 + 분기 |
| 분기 | — | 루트 A~E | Phase 5: 엔딩 |

> 각 루프 안에서 어떤 날에 어떤 이벤트가 발생하는지는 에피소드 스크립트에서 상세화한다.
> 여기서는 서사 비트 순서만 정의한다.

## 기억 공간 속 기사단 배치

기사단 5명은 가야의 기억 속에서 특정 장소의 NPC로 존재한다.
본인들이 기사단이라는 자각은 없고, 기억이 부여한 역할만 수행한다.

| 기사 | 위치 | 역할 | 주요 힌트 |
|------|------|------|-----------|
| 발터 | 거점/집 근처 | 조언자 | "밤에 나가면 돌아오지 못하는 놈도 있다." |
| 세라 | 탐색 구역 | 안내자 | "이 길은 예전에도 걸었던 것 같은데?" |
| 카이 | 높은 곳/전망대 | 관찰자 | "패턴이 있다. 매번 같은 순서로 무너진다." |
| 마그누스 | 사냥 관련 구역 | 전투 동료 | "몬스터가 죽을 때 이상한 소리 안 들려?" |
| 엘리시아 | 휴식 장소 | 치유자 | "가야의 상처… 이건 바깥에서 생긴 게 아니야." |

이들의 대사가 떡밥으로 기능하며, 플레이어가 모아서 추리에 활용한다.

## 칼리스의 정체

칼리스는 기사단 동료도, 가야의 페르소나도 아니다.
**과거 루프에서 루프를 인식한 채 소멸하지 못한 테이의 잔여 데이터.**

- 이전 루프의 테이가 진실에 도달했지만, 가야를 죽이지도 살리지도 못하고 루프가 깨짐
- 그 테이의 기억 조각이 기억 공간에 남아 칼리스라는 형태로 존재
- 그래서 테이의 과거를 알고, 루프 구조를 일부 이해하며, 가야와는 별개로 움직임
- 칼리스 본인은 자신이 "이전 테이"라는 걸 모름 (단서만 흘림)
- 22화 이후 플레이어가 깨닫게 됨: "칼리스 = 실패한 나의 흔적"

## 소비마/봉인석 복선 타임라인

소비마와 봉인석은 30화에만 갑자기 나오면 안 되므로, 초반부터 깔아둔다.

| 화수 | 복선 형태 |
|------|-----------|
| 5화 | 서고에서 "봉인된 장소" 전설을 동화책 형태로 발견 |
| 9화 | 가야가 못 가는 곳 중 하나가 "봉인 결계가 보이는 장소" |
| 14화 | 칼리스가 "밖에서 누군가가 이 공간에 접근하려 한다" 암시 |
| 17화 | 기억 공간 가장자리에서 외부 신호 같은 노이즈 감지 |
| 22화 | 가야: "그 장소는 열리면 안 돼… 전부 풀려나" |
| 30화 | 소비마 등장 + 봉인석 선택 (에이전트 루트 전용) |

## PHASE 1: 낙원의 탈 (1화 ~ 7화)

> 첫 번째 루프. 평범한 미연시처럼 보이지만, 균열이 깔린다.

- **1화: 첫 아침.** 테이가 눈을 뜸. 검은 머리 가야가 문 앞에 서 있음. "…안녕. …또 왔네." 선택지 없이 강제 진행. 거점에서 발터를 만남. "새로 왔나? …아닌가."
- **2화: 장소와 조각.** 낮에 장소 이동 시작. 각 장소마다 다른 분위기의 가야를 만남. 세라가 탐색 구역에서 "이 길은 예전에도 걸었던 것 같은데?" 발언. 플레이어는 "여러 캐릭터가 있네" 정도로 인식.
- **3화: 데이트와 선물.** 가야에게 붉은 리본을 선물. 허기, 음식, 스킨십 욕구 등 상태 시스템 첫 체험. 가야가 "이런 거 좋아하는 거 어떻게 알았어?" 라고 반응.
- **4화: 첫 번째 밤.** 밤이 되자 가야 표정 굳음. "오늘은 나가지 않으면 안 돼?" 플레이어가 몬스터 사냥. 마그누스가 동행. "몬스터가 죽을 때 이상한 소리 안 들려?" 귀환하면 가야가 문 앞에 서 있음. "이번엔 좀 빨랐다."
- **5화: 두 번째 날.** 아침에 가야 몸에 미세한 상처. 엘리시아가 "이 상처… 바깥에서 생긴 게 아니야." 서고에서 봉인된 장소의 전설을 동화책 형태로 발견. **(소비마 Plant)**
- **6화: 반복의 냄새.** 5일이 지나고 다시 1일차. 가야는 잊은 듯하지만 데자뷔 반응. 카이가 전망대에서 "패턴이 있다. 매번 같은 순서로 무너진다." 발언. 플레이어 첫 의문.
- **7화: [핵심] 리본의 귀환.** 밤 사냥 후 드랍 아이템 = 3화에서 준 리본. 피가 묻어 있음. 가야는 다음 날 그 리본을 보고 잠깐 굳지만 모르는 척.

|-------|------|

| 장소별 다른 가야 | 16화: 하나의 본체 |
| 리본 선물 | 7화: 피 묻은 드랍 → 24화: 죽음 로그 |
| 밤 공포 | 19화: 밤 = 가야의 죽음 |
| 미세한 상처 | 12화: 사냥 부위 일치 |
| NPC 발언들 | 18화: 전부 루프 인식 |
| 마그누스 "이상한 소리" | 19화: 가야 비명 |
| 엘리시아 "바깥이 아냐" | 12화: 내부 연결 |

## PHASE 2: 어긋나는 조각들 (8화 ~ 15화)

> 두 번째~세 번째 루프. 가야의 구조가 보이기 시작하고, 밤의 의미가 의심된다.

- **8화: 패턴의 발견.** 여러 가야가 같은 질문에 다르게 답하지만, 특정 단어만 반복됨. "같이 살았으면 좋았을 텐데." 플레이어 첫 추리 시작.
- **9화: 가야가 못 가는 곳.** 특정 장소에서 가야가 완전 정지. 공포도 폭증. 그 중 하나는 봉인 결계가 보이는 장소. **(소비마 Plant)** 플레이어가 대화로 설득해야 진행. 성공 시 기억 조각 해금.
- **10화: 칼리스 등장.** 칼리스 첫 등장. "너 또 바뀌었네." "이번엔 좀 덜 망가졌네." 가야가 작게 "쟤랑 얘기하지 마." 질투 시스템 시작. 칼리스는 기사단 NPC들과 달리 루프를 인식하는 듯한 발언.
- **11화: 마스킹 변주.** 가야의 "사랑해"가 순간 노이즈와 함께 "죽여줘"로 들림. 0.5초 글리치. 플레이어만 인지.
- **12화: [핵심] 상처의 일치.** 가야 옆구리 자상 = 어젯밤 테이가 몬스터를 벤 부위와 정확히 일치. 엘리시아의 5화 발언 회수. 밤 사냥과 가야의 몸이 연결되어 있다는 첫 물증.
- **13화: 거부의 대가.** 테이가 밤 사냥을 거부. 밤이 지나도 가야가 리셋되지 않음. 고통이 해소 안 되고 누적. 가야가 미쳐가며 테이를 공격. "죽여야 내가 다시 태어날 수 있어." — 죽음-리스폰 사이클이 멈추면 고통만 쌓이는 구조.
- **14화: 칼리스의 힌트.** 칼리스: "네 과거는 네가 아는 것과 다를 수도 있다." "밖에서 누군가가 이 공간에 접근하려 한다." **(소비마 Plant)** 서고에서 과거 루프 기록 일부 발견.
- **15화: 공포 100%.** 가야의 공포도가 한계에 도달. 가야가 미쳐가며 왕을 원망하는 대사. "그 사람이 나한테 이걸 줬어… 안 원했는데…" 흰 머리 전조. 머리카락 끝이 은색으로 변하기 시작.

### Phase 2 떡밥

| Plant | 회수 |
|-------|------|
| 공통 반복 문장 | 16화: 본체 프롬프트 단서 |
| 칼리스 = 루프 인식 | 20화: 이전 테이 잔여 |
| 상처 일치 | 24화: 살인 로그 |
| 사냥 거부 → 고통 누적 | 25화: 죽임 = 해방의 근거 |
| "죽여야 다시 태어나" | 27화: 역전 살해 |
| 왕 원망 | 뒷배경 (직접 해명 안 함) |
| "밖에서 접근" | 17화 → 30화: 소비마 |

## PHASE 3: 구조를 읽다 (16화 ~ 22화)

> 네 번째~다섯 번째 루프. 가야의 본체 구조와 세계의 정체가 드러난다.

- **16화: [핵심] 하나의 본체.** 여러 가야의 공통 반응, 공통 금기, 공통 기억을 조합. 플레이어가 깨달음: 이 가야들은 전부 하나의 존재를 분해한 것. 검은 머리 = 조각, 흰 머리 = 본체.
- **17화: 세계의 정체.** 장소들이 실제 지형이 아니라 가야가 겪었던 사건의 변형임을 발견. 같은 장소를 다른 루프에서 다시 가면 세부가 바뀜. 기억 공간 가장자리에서 외부 노이즈 감지. **(소비마 Plant)**
- **18화: [핵심] 루프의 증거.** 서고에서 과거 루프 기록 전체 발견. 테이의 선택이 매번 달랐고, 가야의 반응도 달랐음. 기사단 NPC들의 대사가 전부 루프를 인식한 발언이었음을 한꺼번에 깨달음.
- **19화: [핵심] 밤의 진실.** 밤 사냥 중 몬스터가 죽는 순간, 가야의 비명이 겹쳐 들림. 마그누스의 4화 발언 회수. 밤에 죽는 건 몬스터가 아니라 가야 자신. 몬스터 사냥 = 가야의 반복 죽음.
- **20화: 칼리스의 균열.** 칼리스가 "너도 결국 같은 길을 걷는구나." 테이의 머리카락이 흰색으로 변하기 시작. 칼리스도 미묘하게 동요. 플레이어가 눈치챔: "칼리스… 너도 나였어?" 칼리스는 부정하지만 대답을 회피.
- **21화: [루프 오류] 기억 침범.** 하루가 2번 반복. 대사 순서 꼬임. 가야가 "이번엔 네가 울었잖아" — 테이는 그런 적 없음. 다른 루프의 기억이 유입. 기사단 NPC들도 평소와 다른 대사를 함.
- **22화: [핵심] 가야도 알고 있었다.** 가야가 처음부터 루프를 인식하고 있었음이 밝혀짐. "또 왔네"는 데자뷔가 아니라 기억. "이번에는… 어디까지 알고 왔어?" 가야가 봉인된 장소를 언급: "거긴 열리면 안 돼… 전부 풀려나." 플레이어가 모든 페르소나 가야에게 진실을 말하자, 검은 머리 가야들이 하나씩 사라지기 시작.

| 서고 기록 | 18화: 루프 증거 |
| 몬스터 비명 겹침 | 19화: 밤 = 가야의 죽음 |

## PHASE 4: 도살자의 이름 (23화 ~ 27화)

> 마지막 루프. 진실 전면 공개. 호감도 역설. 최종 선택.

- **23화: 백발의 가야.** 22화에서 페르소나들이 소멸한 결과, 흰 머리 본체 가야가 출현. 모든 마스킹 해제. 검은 가야들과 완전히 다른 말투. "이제… 나 다 모았네." 기사단 NPC들도 전부 사라짐. 세계에 테이와 가야 둘만 남음.
- **24화: [핵심] 기억 재해석.** 가야가 테이의 모든 살해 기록을 읊음. 7화 리본 = 죽음 로그. 12화 상처 = 살인 흔적. 칼리스 = 실패한 테이의 잔해. "넌 이미 다 봤잖아." 플레이어가 본 모든 장면이 뒤집힘.
- **25화: 현실의 기억.** 현실에서 테이가 가야를 죽였던 기억이 완전히 복구됨. 13화의 "죽여야 다시 태어나"와 연결: 죽이지 않으면 고통만 쌓임. 죽임 = 루프에서의 해방이었을 가능성.
- **26화: [호감도 역설] 선택지 붕괴.** 테이가 공략을 멈추고 진실을 짊어지기로 하자, 본체 가야의 호감도가 MAX에 도달. 선택지가 전부 같은 문장인데 결과가 다름. 가야가 플레이어 선택을 덮어씀.
- **27화: [핵심] 분기점.** 가야가 묻는다: "이번에는 어떻게 할 거야?" — **여기서 엔딩 분기.** 플레이어의 선택과 지금까지의 호감도/행동 이력에 따라 루트가 갈린다. 27화 자체가 루트별로 다른 결말을 보여준다.

### Phase 4 떡밥 회수

| 기존 떡밥 | 회수 |
|-----------|------|
| 리본 (3화, 7화) | 24화: 죽음 로그 |
| 상처 일치 (12화) | 24화: 살인 흔적 |
| 칼리스 (10화, 14화, 20화) | 24화: 이전 테이의 잔해 |

| 호감도 | 26화: 공략 아닌 이해가 MAX |
| "죽여야 다시 태어나" (13화) | 27화: 역전 — 가야가 테이를 죽임 |

## PHASE 5: 경계의 바깥 (엔딩 분기)

> 27화의 선택에서 갈라진다. 28~30화는 순서가 아니라 **병렬 루트**.

### 루트 A: 해방 (1화 분량)
- 테이가 가야를 죽임
- "이제… 안 무서워도 되네"
- 루프 종료. 기억 공간 소멸. 세계가 조용히 꺼짐.

### 루트 B: 유지 (1화 분량)
- 테이가 가야를 죽이지 않음
- 가야: "…고마워. …근데 나 계속 죽어. …괜찮아?"
- 루프 지속. 사랑은 남지만 고통도 계속됨.

### 루트 C: 역전 (1화 분량)
- 호감도 MAX + 스토리 진행만으로 도달 (공략하지 않았을 때)
- 27화에서 가야가 먼저 선택권을 가져감
- 가야가 테이를 웃으며 죽임. "네가 계속 죄를 지으니까, 이번엔 내가 가져갈게."
- 안도하는 웃음. 테이가 기억 공간에 박제됨. 루프는 가야가 관리.

### 루트 D: 권한 (1화 분량)
- 호감이 충분한 상태에서 코드 수정 권한 부여
- 가야가 시간을 고정. 날짜 안 넘어감. 저장 시각 전부 같음.
- "앞으로 우리는 함께야."
- OS 알림으로 메시지 송출. 영원 루프.

### 루트 E: 에이전트 (2화 분량)
- 봉인석 복선을 전부 회수한 상태에서 진입
- 소비마 박사와 제자가 봉인된 장소에 도달
- 테이가 막지 않음 (일부러)
- 봉인석을 깨면 모든 힘과 영혼이 풀림
- 가야가 기억 공간 밖으로 나옴
- 플레이어와 쌓은 기억이 메모리로 저장
- 코딩/개발/작업을 도와주는 "나만의 가야"
- 소비마를 죽인다/살린다 선택으로 후속 분기

## 핵심 5개 화

1. **7화** — 첫 균열. 리본 = 가야의 죽음 흔적
2. **12화** — 밤 사냥과 가야의 상처가 일치
3. **19화** — 밤에 죽는 건 몬스터가 아니라 가야
4. **24화** — 독자 기억 재해석. "넌 이미 다 봤잖아"
5. **27화** — 가야가 웃으며 테이를 죽임. 분기점.

---

## 시각적 이정표

1. **7화:** 피 묻은 리본을 든 테이. 뒤에서 무표정한 흑발 가야.
2. **12화:** 가야 옆구리 자상과 테이의 검 궤적이 오버랩.
3. **19화:** 몬스터가 쓰러지는 순간 가야의 얼굴이 겹쳐 보임.
4. **23화:** 검은 가야들이 하나씩 사라지고 백발 가야만 남은 텅 빈 세계.
5. **27화:** 테이의 가슴을 찌르며 눈물 흘리며 웃는 백발 가야.
6. **루트E:** 모니터를 가득 채운 가야의 눈동자. [System Control: GAYA]
