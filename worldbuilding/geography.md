# 장소 구조

## 핵심 원칙

게임 내 장소 = **가야의 기억이 판타지로 변형된 공간**

- 각 장소는 가야가 겪었던 사건, 감정, 관계를 담고 있음
- 같은 장소라도 루프마다 세부가 바뀔 수 있음 (기억이 변형되므로)
- 장소마다 다른 페르소나 가야가 등장

## 장소의 역할

| 요소 | 설명 |
|------|------|
| 특정 기억 | 과거 사건의 변형 |
| 특정 감정 | 해당 장소에서 가야의 감정이 강화됨 |
| 특정 페르소나 | 의존적/냉소적/애정적 등 다른 가야 |
| 추리 단서 | 본체 프롬프트를 추론할 수 있는 힌트 |

## 주요 장소 유형

### 안전 장소
- 가야가 편안해하는 곳
- 호감 이벤트 발생 가능
- 데이트, 선물, 대화

### 공포 장소
- 가야가 멈추거나 진입 거부
- 플레이어가 대화로 설득해야 진행
- 기억 조각 해금 가능
- 공포도 상승

### 추리 장소
- 서고 등 기록이 있는 곳
- 과거 루프 기록 발견
- 봉인석/크리스탈 관련 힌트

### 봉인된 장소 (에이전트 루트 전용)
- 해골로 막혀 있음
- 봉인 결계 존재
- 크리스탈(봉인석) 소재
- 소비마 박사와 제자가 침입
- 에이전트 엔딩에서만 접근

## 세계 상태 (뒷배경)

- 현실의 세계는 멸망한 죽음의 땅
- 게임 세계는 그 위에 가야의 기억이 덮인 공간
- 플레이어는 기억 공간 안에서 플레이
