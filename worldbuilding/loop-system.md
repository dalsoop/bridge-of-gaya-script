# 루프 / 밤 / 죽음 구조

## 루프 구조

- 5일 루프
- 각 일자: 아침 / 점심 / 저녁 / 밤
- 낮: 장소 이동, 대화, 추리, 데이트, 상태 교류
- 밤: 플레이어 조종, 몬스터 사냥

## 밤의 본질

- 몬스터 사냥 = 가야의 죽음 반복
- 가야는 매일 밤 죽음을 경험
- 기억 일부 유지 → 공포 누적
- 신체 손상 진행 (날이 갈수록 망가짐)

## 가야가 밤을 두려워하는 이유

- 밖에 있는 몬스터들을 죽이다가 자신이 죽고 다시 스폰됨
- 이유를 끝까지 말하지 않음
- 가야에게 밤은 지옥

## 라토 왕과 신의 연결

- 왕은 매년 달이 가장 밝을 때 신을 죽임
- 신 처치 후 약해진 상태에서 공격받음
- 그 힘이 가야에게 넘어감 → 머리카락 흰색 변화

## 계승과 루프

- 왕의 힘 = 루프 유지 권한 + 죽음 계승
- 가야가 이 힘을 받음 → 죽어도 다시 돌아옴
- 루프를 끊으려면 가야를 죽여야 함
- 또는 크리스탈(봉인석)을 통한 구조적 해결 필요
