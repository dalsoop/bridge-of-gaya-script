# 루프 / 밤 / 죽음 구조

## 루프 구조

- 5일 루프, 반복됨
- 각 일자: 아침 / 점심 / 저녁 / 밤
- 낮: 장소 이동, 대화, 추리, 데이트, 상태 교류
- 밤: 플레이어 조종, 몬스터 사냥

## 밤의 본질

- 플레이어(테이)가 밤에 나가서 몬스터를 사냥
- 가야는 집에 남아 있음. 밤을 극도로 두려워함
- **진실**: 밤에 몬스터가 죽는 것 = 가야가 죽는 것
  - 몬스터를 벤 부위와 가야의 상처가 일치 (12화에서 발견)
  - 몬스터가 쓰러질 때 가야의 비명이 겹침 (19화에서 발견)
- 가야는 죽고 다시 스폰됨
- 기억 일부 유지 → 공포 누적
- 신체 손상 진행 (날이 갈수록 망가짐)

## 가야가 밤을 두려워하는 이유

- 테이가 밤에 나갈 때마다 가야는 죽음을 경험
- 이유를 끝까지 직접 말하지 않음
- 가야에게 밤은 지옥
- 공포도가 100%에 도달하면 가야가 미쳐가며 왕을 원망 (뒷배경 연결)

## 루프와 기억

- 루프가 반복될 때마다 가야는 일부 기억을 유지
- 가야는 처음부터 루프를 인식하고 있었음 ("또 왔네")
- 각 루프에서 테이의 선택이 달랐고, 가야의 반응도 달랐음
- 서고에서 과거 루프 기록 발견 가능 (18화)

## 루프 종료 조건

- 가야를 죽이면 루프 종료 (해방 엔딩)
- 죽이지 않으면 반복 지속 (유지 엔딩)
- 코드 수정 권한 부여 시 가야가 시간을 고정 (권한 엔딩)

## 힘의 계승 (뒷배경)

- 왕의 힘 → 가야에게 이양됨
- 이 힘이 가야를 죽어도 돌아오는 존재로 만듦
- 테이도 같은 구조에 들어가고 있음 (머리카락 흰색 변화)
