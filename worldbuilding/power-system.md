# 힘의 체계

## 그릇

- 모든 사람은 **그릇**을 가지고 있음
- (세부 규칙 미정)

## 신의 힘

- 신은 죽어도 1년에 한 번 부활 (달이 가장 밝을 때)
- 무지성 상태로 태어남
- 부활 시 처리하지 않으면 세계 멸망

## 이든의 힘

- 신을 죽여서 얻은 힘
- 5개 나라에 분배
- 시공간 등 영역별로 나뉨

## 크리스탈 (봉인석)

- 동굴 안에 존재
- 이든의 힘 + 영혼이 봉인됨
- **깨지면**: 힘과 모든 영혼이 풀림
- 소비마 박사와 제자가 이걸 꺼내려 함
- 테이가 일부러 막지 않음

## 전투 체계

- 마법 + 검술 병용 가능
- 라토 기사단은 양쪽 모두 사용

## 계승 구조

```
신 → 이든 (살해로 획득)
     → 5개 나라 왕들 (분배)
         → 라토 왕 (시공간)
             → 가야 (왕위 이양)
                 → 테이/플레이어
```
