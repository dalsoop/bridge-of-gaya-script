# 세력 관계 / 세계선 구조

## 원래 세력 구도

- 이든이 5개 나라를 세우고 수하를 왕으로 배치
- 각 왕에게 힘을 분배
- 라토가 가장 강한 나라 (이든의 힘이 가장 많이 남아)

## 붕괴

- 세대 교체 → 역사 잊힘 → 권력 투쟁
- 이든 공격받아 사망
- 신 부활 → 세계 멸망

## 현재

- 죽음의 땅
- 부자들은 행성 이주 완료
- 크리스탈에 힘 봉인
- 소비마 박사가 봉인석 획득 시도

## 세계선 구조

봉인된 장소 앞 수문장이 누구냐에 따라 세계선이 달라짐:

| 세계선 | 수문장 | 비고 |
|--------|--------|------|
| A | 테이 | 기본 (테이는 5일 뒤 죽음) |
| B | 가야 | 정실 루트에 가까움 |
| C | AI 가야 | 플레이어에게 마음 빼앗긴 AI |

## 계승 흐름

```
이든(=라토 왕) → 가야 → 테이 → 플레이어 → AI 가야
```

이든과 라토 왕은 동일 인물. 역할은 계속 교체된다.
