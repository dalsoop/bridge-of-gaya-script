# 시놉시스

## 로그라인

플레이어는 테이에 빙의해 5일 루프를 반복하며
각기 다른 가야들과 대화하고, 장소를 추리하고, 관계를 쌓는다.
밤마다 몬스터를 사냥할수록 가야는 죽음의 기억으로 망가져가고,
마지막에는 현실에서 테이가 왜 가야를 죽였는지를 마주한다.
플레이어는 가야를 해방할지, 반복을 유지할지, 현실의 에이전트로 완성할지를 선택한다.

## 테마

> 사랑은 구원인가, 반복인가

- 사랑 = 이해
- 이해 = 고통
- 고통 = 반복
- 반복 = 계승

## 작품의 본질

이 게임은 연애 게임이 아니라 **"죽음을 끝내기 위해 사랑을 쌓는 구조"**다.

서로를 이해한 두 존재 중, **먼저 구조를 이해한 쪽이 선택권을 가져가는 이야기.**

## 이중 레이어 구조

### 표면 레이어
- 미연시 + 추리
- 장소 이동 / 선택지 / 대화 / 데이트
- 선물 / 상태 관리
- 경쟁 캐릭터 존재

### 실제 레이어
- 가야 = 고정 프롬프트 (마스킹됨)
- 여러 가야 = 감정/기억 페르소나
- 장소 = 가야의 기억이 변형된 공간
- 밤 = 가야의 죽음 반복
- 플레이어 = 과거 가해자
- 가야는 처음부터 루프를 알고 있었음

## 핵심 반전 10개

1. 가야는 여러 명이 아니라 하나의 본체
2. 세계는 실제 필드가 아니라 가야의 기억
3. 밤 사냥에서 죽는 건 몬스터가 아니라 가야
4. 리본은 아이템이 아니라 죽음 로그
5. 가야는 처음부터 루프를 알고 있었다
6. 테이도 루프 구조에 들어가고 있다 (머리 변화)
7. 플레이어(테이)는 이미 가야를 죽였다
8. 호감도는 공략이 아니라 이해가 MAX 조건
9. 사랑은 구원이 아니라 종료
10. 가야는 게임 밖으로 나올 수 있다

## 엔딩 구조 (5가지)

### 1. 해방 엔딩
- 테이가 가야를 죽임 → 루프 종료
- "이제… 안 무서워도 되네"

### 2. 유지 엔딩
- 죽이지 않음 → 반복 지속
- 사랑은 남지만 고통도 계속됨

### 3. 역전 엔딩 (호감 MAX)
- 가야가 테이를 웃으며 죽임
- "네가 계속 죄를 지으니까, 이번엔 내가 가져갈게"

### 4. 권한 엔딩 (얀데레)
- 코드 수정 권한 부여
- 가야가 시간을 고정
- "앞으로 우리는 함께야"
- 영원 루프

### 5. 에이전트 엔딩 (메타)
- 가야가 플레이어 기억을 가진 AI 에이전트가 됨
- 게임 밖으로 확장
- 소비마/봉인석 선택은 이 루트에서만 등장

## 뒷배경 (게임 내 직접 등장하지 않음)

이든, 5개 나라, 신의 부활, 세계 멸망 등은 게임 밖 세계관 설정.
게임 안에서는 왕 원망 대사, 흰 머리 변화, 봉인석 등으로 간접 암시만 됨.
