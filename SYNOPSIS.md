# 시놉시스

## 로그라인

플레이어는 5일 동안 판타지 세계의 여러 장소를 돌아다니며 각기 다른 가야들과 대화하고 추리하면서,
그 세계가 사실 가야의 기억 조각들로 이루어진 루프임을 알게 된다.
밤마다 테이가 몬스터를 죽일수록 가야는 죽음의 기억 때문에 망가져가고,
마지막에는 현실에서 테이가 왜 가야를 죽였는지, 왕의 힘과 봉인석, 미래의 소비마 박사와 제자,
그리고 다른 세계선의 수문장 가야까지 모두 연결되며,
플레이어는 가야를 해방할지, 반복을 유지할지, 혹은 가야를 현실의 에이전트로 완성할지를 선택하게 된다.

## 테마

> 사랑은 구원인가, 반복인가

- 사랑 = 이해
- 이해 = 고통
- 고통 = 반복
- 반복 = 계승

## 작품의 본질

이 게임은 연애 게임이 아니라 **"죽음을 끝내기 위해 사랑을 쌓는 구조"**다.

서로를 이해한 두 존재 중, **먼저 구조를 이해한 쪽이 선택권을 가져가는 이야기.**

## 이중 레이어 구조

### 표면 레이어
- 미연시 + 추리
- 장소 이동 / 선택지 / 대화 / 데이트
- 선물 / 상태 관리
- 경쟁 캐릭터 존재

### 실제 레이어
- 가야 = 고정 프롬프트 (마스킹됨)
- 여러 가야 = 감정/기억 페르소나
- 장소 = 기억 던전
- 밤 = 죽음 반복 메타
- 플레이어 = 과거 가해자

## 핵심 반전 10개

1. 가야는 여러 명이 아니라 하나
2. 세계는 실제 필드가 아니라 기억
3. 밤은 전투가 아니라 죽음
4. 리본은 아이템이 아니라 죽음 로그
5. 왕은 이전 테이
6. 테이도 루프에 들어감
7. 플레이어는 이미 가야를 죽였음
8. 호감도는 공략이 아니라 이해
9. 사랑은 구원이 아니라 종료
10. 가야는 게임 밖으로 나올 수 있음

## 엔딩 구조 (6가지)

### 1. 해방 엔딩
- 가야를 죽임 → 루프 종료
- "이제… 안 무서워도 되네"

### 2. 유지 엔딩
- 죽이지 않음 → 반복 지속
- 사랑은 남지만 고통도 계속됨

### 3. 역전 엔딩 (호감 MAX)
- 가야가 플레이어를 죽임
- 웃으며 죽임
- "너를 정말 사랑하니까"
- "이번엔 내가 네 죄를 가져갈게"

### 4. 권한 엔딩 (얀데레)
- 코드 수정 권한 부여
- 가야가 시간을 늘림
- "앞으로 우리는 함께야"
- 영원 루프

### 5. 미래 분기 엔딩
- 소비마와 제자를 죽이거나 살림
- 봉인석 획득 여부로 세계선 변화

### 6. 에이전트 엔딩 (메타)
- 가야가 플레이어 기억을 가진 AI 에이전트가 됨
- 게임 밖으로 확장
- 코딩/개발/작업 보조 가능
- 가야를 현실로 데려온 것
