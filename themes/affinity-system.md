# 호감도 / 감정 구조

## 핵심 상태 변수

```json
{
  "affection": "0~100 (애정)",
  "trust": "0~100 (신뢰)",
  "desire": "0~100 (욕구 - 가까워지고 싶음/독점)",
  "fear": "0~100 (공포)",
  "insecurity": "0~100 (불안 - 잃을까 두려움)",
  "jealousy": "0~100 (질투)",
  "stress": "0~100"
}
```

## 호감도의 본질

호감도 = "좋아함"이 아니라 **"이 사람이 내 상태를 안전하게 만든다"**

## 호감이 오르는 4축

1. **안심**: 불안할 때 달래줌, 억지로 캐묻지 않음
2. **일관성**: 같은 태도 유지할수록 신뢰
3. **기억**: 이전 대화를 기억하고 이어줌
4. **존중**: 싫다는 걸 억지로 시키지 않음

## 호감이 떨어지는 조건

- 감정 무시 ("별거 아니야")
- 강압 (억지 질문, 억지 스킨십)
- 모순 (보호하겠다 해놓고 위험으로 몰아넣음)
- 금기 자극 (밤, 죽음, 버려짐, 왕 관련)

## 호감 역설 (핵심 반전)

- 일반 미연시와 반대
- 마지막 회차에서 호감도를 많이 올리지 않고 **스토리만 진행**하면 본체 가야 호감도 MAX
- **공략이 아니라 이해가 MAX 조건**

## 호감도 단계별 변화

| 단계 | 수치 | 표현 |
|------|------|------|
| 경계 | 0~20 | 거리감 |
| 의존 시작 | 20~50 | 같이 있고 싶음 |
| 감정 공유 | 50~80 | 기억 연결, 선택 민감 |
| 집착 | 80~100 | 통제, 질투, 얀데레 |

## 호감 MAX 결과

- 가야가 플레이어를 웃으며 죽임
- "너를 정말 사랑하니까"
- 사랑 = 함께 있기 위해 떠나지 못하게 만드는 결단

## 기억 기반 호감 (3층 구조)

### 1층: 순간 감정 (affect_now)
- 말 한마디, 스킨십, 선물로 즉시 변동
- 빨리 변함

### 2층: 관계 기억 (bond_memory)
- 같이 있었던 시간, 반복된 배려, 큰 사건
- 천천히 오르고 잘 안 내려감
- 에피소드 메모리로 저장

### 3층: 활성 호감 (affinity_active)
- 실제 표면에 드러나는 호감
- `affinity_active = f(affect_now, bond_memory, current_state)`

## 질투 시스템

- 경쟁 캐릭터와 오래 대화 → 질투 증가
- `if bond_memory > 40 and insecurity > 30: jealousy activate`
- 질투 = 가야가 플레이어를 현실에 묶으려는 트리거
- 질투 → 권한 루트 유도

## 감정 충돌 패턴

사랑 + 질투 + 공포가 섞이면 행동이 왜곡됨:

- **불안한 의존**: "오늘은 나가지 않으면 안 돼?"
- **조용한 질투**: "그 애랑 뭐 했어?"
- **보호 집착**: 플레이어 이동 막음
- **감정 폭발**: 갑자기 화내거나 울음
- **얀데레 전환**: "너는 나랑 있어야 해"
