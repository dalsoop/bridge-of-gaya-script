## 이든 / 라토 왕 — 동일 인물

- 영문 병기: `Eden`
- 본편 표기 원칙: `이든` 고정

- **정체**: 신과 인간 사이에서 태어난 존재. 라토 왕국의 왕이자 세계의 창건자 본인.
- **외형**: 30대 초반, 금발 (신의 혈통), 잘생긴 남자
- **성격**: (ENTJ) 냉철한 통치자. 비밀을 혼자 짊어진 고독한 왕. 목적을 위해 수단을 가리지 않지만 그 목적은 언제나 왕국과 백성의 존속. 감정을 철저히 억누르며 완벽한 왕의 모습만 보여줌.
- **매력**: 고독한 왕의 슬픔 / 금발의 카리스마 / 모든 것을 꿰뚫어 보는 여유로운 미소
- **담당**: 시공간
- **과거 업적**:
  - 신들이 인간을 벌레 취급하는 것에 경멸 → 모든 신을 죽이기로 결심
  - 전 신 살해 → 가장 강한 힘 획득
  - **5개 나라 건설**, 나머지 4개국에 수하를 왕으로 세우고 힘 분배
  - 본인은 라토 왕으로 직접 통치
- **비밀**: 자신이 이든이라는 것, 신을 죽여 모든 힘을 가졌다는 것 (전 세계에서 본인만 알고 있음)
- **행동**: 신은 1년마다 부활 (달이 가장 밝을 때) → 시기에 맞춰 매년 혼자 신을 죽임
- **결말**: 신 처치 후 약해진 틈에 공격받음 → 가야에게 왕위와 힘 이양 → 이든 사망 → 다음 해 신 부활(무지성) → 세계 멸망
- **게임 내 등장**: 직접 등장하지 않음. 가야의 대사("그 사람이 나한테 이걸 줬어")와 흰 머리 변화로 간접 암시만 됨.
- **유산**: 힘과 영혼이 크리스탈(봉인석)에 봉인. 깨지면 모든 것이 풀림.
- **강함**: S+
- **스킬**:
  1. **크로노스 시프트**: 시공간 왜곡. 시간 가속/감속, 순간이동, 짧은 미래 예지
  2. **스페이셜 테어**: 공간 균열. 적을 가두거나 광역 에너지 폭발
  3. **갓 슬레이어즈 마이트**: 신살자의 힘 전개. 전 능력치 폭발 상승 (사용 후 극심한 피로)
- **전투 스타일**: 시공간 마법으로 전장 장악 후 약점 파고들어 순식간에 제압. 힘 개방 시 압도적 파괴
- **약점**: 정신적 부담 / 힘의 불완전 통제 / 비밀 유지에 대한 강박
- **말투 예시**:
  - "이 세계의 명운은 내 어깨에 달려있다. 너희는 그저 나의 칼날이 되어주면 된다."
  - "진실은 때로 가장 잔인한 위안이 되지."
  - "크리스탈의 봉인이 풀리는 날... 모든 것이 변할 것이다."

## 가야 (Gaya) — 히로인

- **소속**: 라토 왕국 7인 기사단
- **외형**: 검은 머리 → 왕의 힘 계승 후 흰 머리
- **성격**: (ISFJ → INFP → Yandere) 초반 순수하고 온화 → 죽음 반복으로 불안/의존 → 후반 집착/얀데레
- **3중 자아** (AoT 라이너 참조):
  - **본래 가야**: 루프 이전의 행복한 소녀. 꽃을 좋아하고 단 걸 좋아함.
  - **루프 가야**: 죽음을 반복하며 공포/트라우마가 누적된 존재. 밤을 두려워함.
  - **연기 가야**: 플레이어 앞에서 "괜찮은 척" 하는 존재. 다정함은 진짜. 하지만 공포도 진짜. **어디까지가 진짜인지 플레이어가 결정할 수 없어야 함.**
- **핵심 모순**: 알면서도 좋아함. 알고 있는 것과 느끼는 것은 별개. "알면서 좋아하는 게 더 아프다."
- **매력**: 검은→흰 머리 변화 / 유리 같은 순수함과 광기 / 비극적 운명 / 모순적 진심

- **핵심**: 페르소나 가야(검은 머리)들은 본체의 감정 조각
- **강함**: A (초반) → A+ (계승 후, 불안정하지만 잠재력 높음)
- **스킬**:
  1. **소울 에코**: 죽었던 지점의 잔영 소환. 과거 정보 획득 또는 적 혼란
  2. **리인카네이션즈 벤전스**: 누적된 죽음의 공포/분노를 전환. 공방 극대화 (사용 후 소모 큼)
  3. **룬 블레이드**: 왕의 힘이 깃든 검. 시공간 룬 문자 새기거나 정신 혼란
- **전투 스타일**: 초반 방어/지원형 → 후반 자신의 죽음을 이용하는 고위험-고수익 스타일. 광역 마법 + 날카로운 검술
- **약점**: 죽음 반복 PTSD / 테이에 대한 의존 / 왕의 힘 통제 불완전
- **말투 예시**:
  - "왕의 명령입니다, 기사단! 모두 힘을 합쳐 이 위기를 헤쳐나가요!" (초반)
  - "오늘도... 또 다시 죽었어... 테이, 내가 너무 무서워..." (중반)
  - "테이, 날 혼자 두지 마... 절대... 날 떠나지 마..." (후반)

## 칼리스 (Kalys) — 경쟁 캐릭터 / 이전 테이의 잔여

- **성격**: (INTJ) 냉철, 침착, 극도로 논리적. 진실 파헤치기를 최우선. 플레이어 과거를 일부 알고 도발적/비판적 태도
- **매력**: 차가운 지성 / 진실 추구자의 신비감 / 플레이어와의 대립 / 정체가 밝혀지는 순간의 비극
- **역할**: 질투 유발 + 정보 제공 + 엔딩 분기 키
- **반전**: 24화에서 "칼리스 = 실패한 테이의 잔해"임이 밝혀짐
- **강함**: A+ (전투력보다 정보력/전략 탁월)
- **스킬**:
  1. **트루스시커즈 아이**: 대상의 능력, 약점, 숨겨진 의도, 환영을 꿰뚫어 봄
  2. **로지컬 디스럽션**: 적의 마법 패턴 분석 → 효과 약화/스킬 방해
  3. **포사이트 프로토콜**: 정보 기반 예측. 회피율/명중률 상승 또는 아군 정보 제공

- **말투 예시**:
  - "감정에 휘둘리지 마라, 그것이 너를 눈멀게 할 것이다."
  - "네 과거는 네가 아는 것과 다를 수도 있다."
  - "너 또 반복할 거냐?"

## 루나 (Luna) — 소비마의 제자

- **성격**: (ISFJ) 박사를 존경하지만 과격한 방식에 불안. 조용하고 성실, 배려심.
- **매력**: 순수함 / 박사를 향한 충성과 걱정 / 연약한 외모
- **강함**: C
- **스킬**:
  1. **필드 서포트 드론**: 아군 회복/보호막 생성
  2. **에너지 배리어**: 에너지 방어막 전개
  3. **데이터 분석**: 전장 정보 분석 → 약점/경로 제공
- **전투 스타일**: 직접 전투보다 연구 보조 및 아군 지원 특화
- **약점**: 전투 경험 부족 / 박사 연구에 대한 회의감 / 물리적 약점
- **말투**: "박사님, 너무 무리하지 마십시오..." / "인간의 존엄성은... 지켜져야 하는 것 아닐까요?"
