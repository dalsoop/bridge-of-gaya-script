# 다섯 나라

## 엑산 — 불

| 항목 | 설정 |
|---|---|
| 원소 | 불 |
| 머리색 | 붉은빛 (짙은~옅은 스펙트럼) |
| 세례 의식 | 용광로 앞 무릎. 불이 몸을 스치며 돌이 손에 놓임 |
| 지리 | 용광로 도시, 재 폐기장, 성벽 |
| 사회 | 노비촌·수호인·왕실 3층. 계급 엄격 |
| 특산 | 철 · 원석 (실은 크리스탈 원재료, 1부 은폐) |
| 이든의 출신지 | 노비촌 |

### 엑산의 그림자
- 성벽은 하루 두 번 붉어진다. 해 뜰 때 · 해 질 때
- 공식: 용광로 반사광
- 민담: 성벽 안쪽에서 타오르는 불의 영혼이 돌을 뚫고 배어 나온다
- **숨은 진실**: 용광로 바닥에 대형 원석 (크리스탈) 매장

## 시아바 — 물

| 항목 | 설정 |
|---|---|
| 원소 | 물 |
| 머리색 | 푸른빛 |
| 세례 의식 | 성수에 몸을 담그는 순간 돌이 물 아래로 가라앉으며 선택됨 |
| 지리 | 해안·강·호수. 물의 도시들 |
| 사회 | 상업·해상 무역 중심. 상대적으로 유연 |
| 특산 | 수산물, 염료, 수정 세공 |

## 레탄 — 바람

| 항목 | 설정 |
|---|---|
| 원소 | 바람 |
| 머리색 | 은회색 |
| 세례 의식 | 높은 탑 꼭대기. 바람 속에서 돌이 손바닥에 얹힘 |
| 지리 | 고산, 고원, 탑의 도시 |
| 사회 | 점술·예지 문화 강함. 노인 지혜 중시 |
| 특산 | 희귀 약초, 날짐승 |

### 레탄 민담
- *"바람이 가야 쪽으로 가면 한 번 접힌다"* — 레탄 노인들이 흘리는 말

## 솔라니 — 흙

| 항목 | 설정 |
|---|---|
| 원소 | 흙 |
| 머리색 | 연갈색 |
| 세례 의식 | 동굴 제단에서 돌을 직접 캐내듯 받음 (유일 능동형) |
| 지리 | 광산, 동굴, 석조 왕궁 |
| 사회 | 장인·세공 문화 최강. 부유 |
| 특산 | 보석, 광물, 조각 |

### 솔라니의 그림자
- 왕궁·왕관·귀족 의복까지 전역이 **보석 치장**
- 왕 타카의 농담: *"이 나라는 돌과 대화한다"*
- **숨은 진실**: 크리스탈 가공·관리의 중심지. 솔라니 사제는 시스템을 알고 있다

## 라토 — 시공간 (쇠퇴)

| 항목 | 설정 |
|---|---|
| 원소 | 시공간 |
| 머리색 | 흰색 |
| 세례 의식 | 소실됨. 집시들 말로만 남음 |
| 지리 | 폐허·집시 유랑지. 중앙에 **가야의 다리** |
| 사회 | 국가 구조 붕괴. 집시·점쟁이·떠돌이 |
| 특산 | 짧은 순간이동, 흐릿한 예지 (집시들에게만 잔존) |

### 라토의 진실
- 공식: 왕 실종, 민족 흩어짐
- 민담: *"라토의 마지막 밤엔 보석이 다 하늘로 날아갔다"*
- **숨은 진실**: 대수확 사건으로 의도적으로 희생됨. 중앙 크리스탈은 가야의 다리 아래
- 라토가 다시 서는 순간 = 세계 균형이 흔들리는 순간

## 네 나라의 관계

- 네 나라(엑산·시아바·레탄·솔라니)는 외교·교역·소규모 분쟁의 균형
- 라토 공백은 모두가 암묵적으로 비워 두고 있음 — 이유는 각 나라 왕실만 안다
- 1부 진행과 함께 이든이 이 균형을 위협 → 4왕이 각자 이든을 찾아온다
