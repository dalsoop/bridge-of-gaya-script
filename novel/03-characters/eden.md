# 이든

## 기본 정보

- **이름**: 이든 (1부 Ch3/Ch4 자가 명명 시점까지는 "까마귀"로만 불림)
- **나이**: 16세 (1부 시작 시점)
- **출신**: 엑산 왕국 노비촌
- **계급**: 노비 → 1부 종결 시 라토의 왕
- **머리색**: 검정 (태어날 때) → 끝자락 연노랑 → 절반 연노랑 → 거의 전체 흰-금 (22장)
- **능력**: 시공간 접기·순간이동·시간 짧게 되돌리기

## 정체

### 공식판 인식 (이든 본인·주변인)
- 엑산 노비촌에서 태어난 이름 없는 아이
- 엑산 붉은 계열 머리에 속하지 않아 "까마귀"라 불림
- 노비촌 최하층

### 진짜 정체 (집필용 정본)
- **시공간 신의 친아들**
- 시공간 신은 리쿠스의 자식 (반신)
- 이든은 그 반신의 아들 → 계보상 리쿠스의 손자뻘
- 친모는 불명 (2부에서 공개 가능)
- 이든 본인은 자기 정체를 Ch3 리쿠스와의 만남에서 부분적으로 안다
- 전모는 1부에서 완전 공개되지 않음

## 시스템적 위치

- **세례 밖** — 이 행성에서 세례 없이 능력을 쓰는 유일한 인간
- **크리스탈 링크 없음** — 능력을 써도 수명·기억이 지불되지 않음
- **두 개의 영혼** — 타카토의 눈엔 한 몸에 두 자아가 겹쳐 보임

## 능력 성장 (1부)

| 장 | 그릇 |
|---|---|
| Ch3 | 3 걸음 (첫 시도) |
| Ch5 | 열 걸음 (한계 발견) |
| Ch5 끝 | 서른 걸음 (경계 돌파) |
| Ch10 | 수십 걸음, 시간 짧게 되돌림 |
| Ch14 | 수백 걸음 단위 |
| Ch19 | 나라 사이 이동 가능 |
| Ch22 | 반경 100걸음 공간 접기 + 수호신 조련 |

## 성격

- 말수 적음 (노비촌에서 배운 생존 기술)
- 감정 표현 억제
- 타인에 대한 기본 불신 + 이따금의 예외적 친절 (바이후에게)
- 고통에 대한 높은 내성
- 자기 이상을 천천히 받아들이는 타입 (패닉 없음)

### 성장에 따른 변화
- Ch1~4: 외부 자극에 거의 반응 없음
- Ch5~10: 자기 힘을 스스로 실험·측정 시작
- Ch11~14: 책임감의 발아 (바이후·도로 마을 보호)
- Ch15~19: 왕이 된다는 것의 무게
- Ch20~22: 침묵하면서도 결정권자 — 말보다 존재로 결정

### 먼치킨/사이다 톤
- 전투에서 힘을 자랑하지 않음
- 압승을 당연하게 처리
- *"건드리지 마. 오늘부터."* 같은 짧은 선언
- 공포나 분노보다 **정적**으로 상대를 무너뜨림

## 중요 관계

### 리쿠스 (스승·의문의 존재)
- Ch1~4에서 능력 각성 안내
- Ch5부터는 이든 앞에 덜 나타남
- 이든은 리쿠스를 전적으로 믿지 않는다 (본능적 거리)

### 바이후 (친구)
- 도로 마을 출신
- 1부 유일한 정서적 지지 관계
- 능력 없음
- 이든의 왕이 된 뒤에도 끝까지 친구로 남는다

### 박씨 (관리자)
- 엑산 노비촌 관리자
- 이든을 때리는 사람이었으나 이든의 능력을 눈치챈 뒤 태도 변화
- 박씨의 진짜 정체: 2부에서 재등장 여지 (라토 잔존 조력자 가능성)

### 타카토 (영혼을 보는 자)
- Ch7에서 처음 만남
- 이든의 "두 개의 영혼"을 알아본 유일한 인물
- 타카토 자신도 시스템 밖에 있는 인종

### 네 왕 (시아바·레탄·솔라니·엑산)
- Ch16~19에서 순차 회합
- 각자 이든에 대한 계산 다름
- 공통: 모두 이든을 두려워한다

### 가야 (2부)
- 1부 동안은 존재 자체가 숨겨짐
- 이든이 Ch22 수호신화를 완성하는 순간 다리에서 태어남
- 2부에서 처음 만남

## 이든의 금기·취약점

- 지속적 물리적 부상에는 일반인처럼 약함
- 수면 중엔 능력 사용 불가
- 감정적 동요 심할 때 시공간 접기 정확도 떨어짐
- 본인 능력의 근원을 모를 때 그릇이 답보 — 리쿠스가 이를 이용해 성장을 유도

## 외형 묘사 포인트

- 검정 머리 → 연노랑 → 흰-금 진행이 서사의 시각 축
- 왼쪽 어깨 — Ch2 꿈에서 리쿠스가 손을 얹었던 자리. 가끔 뜨거워짐
- 손 — 1부 후반 피가 묻기 시작함 (씻어도 지워지지 않는다는 예언의 회수)
- 왕 시점(프롤로그)의 외형 = 흰-금 머리 + 끝자락 검은 한 가닥 + 피 묻은 손
