# EP.03 — 데이트와 선물

> **루프 1회차 / Day 3. 첫 데이트. 리본 선물. 가야와 거리가 좁혀지는 breather.**

## 개요
가야와 첫 데이트. 선물/스킨십/상태 시스템 본격 등장. 리본 선물이 7화 핵심 떡밥의 시작점. 가장 평화로운 에피소드 — 나중에 더 아프게 하기 위한 "감정 예금".

## 등장인물
- 테이, 가야 (기본 페르소나), 엘리시아

## 씬 구성

### S1. 거점 — 아침
```yaml
scene_id: ep03_s1
location: 거점_방
time: 아침
bgm: morning_warm_01
tension: 2
pacing: breather
primary_emotion: warmth
```

가야 (밝게): "오늘은… 어디 갈래?"

- 데이트 장소 선택지 등장 (안전 장소 2~3개)
- **숨겨진 취향 시스템 작동 시작 (마스킹된 프롬프트의 일부)**
- 가야가 특정 장소를 고르면 표정이 밝아짐 → 취향 힌트

### S2. 데이트 장소 — 오전~점심
```yaml
scene_id: ep03_s2
location: 데이트_꽃밭 (또는 선택한 장소)
time: 오전~점심
bgm: date_01
tension: 2
pacing: breather
primary_emotion: warmth
romance: push
intimacy: 2→3
```

- 가야와 둘이서 산책
- **[매력 비트 1: 가야의 일상]**
- 가야가 꽃을 보며 멈춤. 무의식적으로 손을 뻗어 만지다가 테이가 보고 있는 걸 눈치챔.

가야 (당황, 손 뒤로): "…아, 아니야. 그냥."

가야 (작게, 시선 내림): "…예쁘니까."

- **이 순간이 가야를 "설정"이 아니라 "사람"으로 느끼게 만드는 첫 비트.**

- 대화 선택지 다수. 가야의 반응에서 취향/금기 힌트가 슬쩍 나옴:

가야 (눈 반짝): "단 거 좀 땡겨…"

가야 (움찔): "시끄러운 건 싫어."

- **[매력 비트 2: 가야만의 습관]**
- 가야가 걸으면서 무의식적으로 머리카락을 귀 뒤로 넘김.
- 음식을 먹을 때 한쪽 볼만 부풀림.
- 뭔가 좋으면 발끝이 살짝 들림.
- **플레이어가 이 습관들을 알아차리게 되면 가야에 대한 관심이 올라간다.**

- 특정 화제를 꺼내면:

가야 (멈춤, 0.5초): "……"

가야 (빠르게 복귀): "아니야, 그건 됐어."

- **금기 단서. character-voice의 banned_words 영역.**

- **[매력 비트 3: 불시의 진심]**
- 바람이 불어 가야의 머리카락이 날림. 가야가 테이를 올려다봄.

가야 (바람에 머리 날리며, 작게 웃음): "…이런 날이 좋아."

가야 (시선이 멀어지며): "…누군가랑 이렇게 걸었던 적이… 있었던 것 같은데."

가야 (고개 흔들며): "…모르겠다."

- **기억 조각의 잔재. 루프 이전의 행복한 기억이 스며나옴. 가야가 단순 NPC가 아님을 느끼게 함.**

- 스킨십 시스템 소개: 손잡기 제안 가능
  - 수락 시:

가야 (얼굴 붉힘, 손을 잡고): "…어."

가야 (작게, 손에 힘을 줌): "…따뜻해."

가야 (한참 후, 더 작게): "…놓지 마."

  - 거절 시:

가야 (고개 숙임): "…아직은 좀…"

가야 (하지만 테이 옆으로 반 걸음 더 가까이): "…근데 이 정도는 괜찮아."

  - **거절해도 "반 걸음"으로 자기 방식의 호감 표현. 존중에 대한 호감 상승.**

### S3. 선물 — 점심
```yaml
scene_id: ep03_s3
location: 데이트_상점
time: 점심
bgm: date_01
tension: 2
pacing: setup
primary_emotion: warmth
```

- 선물 아이템 구입 가능
- **붉은 리본** 구매 가능
- 가야에게 선물하면:

가야 (눈 커짐): "…이거… 예쁘다."

가야 (작게): "이런 거 좋아하는 거 어떻게 알았어?"

- 가야가 리본을 머리에 묶음
- 호감 상승
- **이 리본이 7화에서 피 묻은 채로 돌아온다. Layer 3 심기.**

### S4. 휴식 장소 — 오후
```yaml
scene_id: ep03_s4
location: 휴식장소_엘리시아
time: 오후
bgm: rest_01
tension: 3
pacing: setup → mild escalation
primary_emotion: warmth → concern
```

- 엘리시아가 있는 장소에 잠깐 들름

엘리시아 (미소): "여기서 쉬어가도 돼요. 좋은 사람이랑 같이 왔네."

- 가야에게:

엘리시아 (부드럽게): "가야… 요즘 잘 자?"

가야 (거짓말): "…네. 괜찮아요."

- 엘리시아가 테이에게 따로:

엘리시아 (낮은 목소리): "가야 상태 좀 봐줘. 뭔가… 이상해."

- **엘리시아가 가야의 이상을 눈치채고 있다는 첫 힌트. L8 NPC 대사 Plant.**

### S5. 거점 — 저녁
```yaml
scene_id: ep03_s5
location: 거점_방
time: 저녁
bgm: evening_warm_01
tension: 3
pacing: setup → escalation
primary_emotion: warmth → melancholy
romance: push
intimacy: 3 유지
```

가야 (미소): "오늘… 좋았어."

가야 (조용히): "이런 날이 계속되면 좋겠다."

가야 (작게, 시선 내림): "…계속 될 리 없지만."

- 선택지:
  1. "왜 안 되는데?" → flags: [questioned_future]
  2. "나도 좋았어." → gaya_comfort+2
  3. (가만히 있기) → gaya_comfort+1 (침묵 존중)

- 1번:

가야 (멈춤): "…아니야. 그냥."

- 대화 전환. 하지만 표정이 잠깐 어두워짐.
- **"계속 될 리 없지만" = 가야가 루프의 끝을 알고 있다. L2 "또" 레이어 강화.**

### S6. 방 — 밤
```yaml
scene_id: ep03_s6
location: 거점_방
time: 밤
bgm: night_gentle_01
tension: 3
pacing: breather → setup
primary_emotion: warmth → mild unease
```

- 밤이 됨
- 가야가 리본을 만지작거리며 앉아 있음

가야 (작게 미소): "…이거 소중히 할게."

- 밤 공포 반응이 전보다 약간 덜함 (데이트 효과 = 감정 예금)
- 하지만 여전히 불안한 기색

가야 (작게): "…내일도 이러면 좋겠다."

[연출] 화면 암전. 텍스트: "DAY 3 END."

- **이 에피소드 전체가 "감정 예금". 7화에서 리본이 피 묻어 돌아올 때 "인출"되는 감정의 원금.**
