# EP.16 — 하나의 본체

> **루프 5회차 / Day 1 (핵심 화)**

## 개요
플레이어가 축적된 추리 결과를 조합하여 깨달음: 여러 가야는 하나의 본체를 분해한 것. 검은 머리 = 조각, 흰 머리 = 본체.

## 등장인물
- 테이, 가야 (복수 페르소나)

## 씬 구성

### S1. 거점 — 아침
- 5회차 시작. 루프마다 쌓인 추리 로그를 정리하는 느낌.
- 가야: "…안녕." (데자뷔도 없는 초기화 상태)
- 하지만 테이는 이미 많은 것을 알고 있음

### S2. 검증 탐색 — 오전~점심
- 플레이어가 의도적으로 모든 장소를 순회
- 각 페르소나 가야에게 동일한 핵심 질문:
  - "넌 누구야?"
  - "다른 가야 알아?"
  - "넌 왜 여기 있어?"
- 반응 패턴:
  - 의존 가야: "나? 그냥… 나인데. 다른 가야가 뭐야?"
  - 냉소 가야: "…그런 질문은 의미 없어."
  - 활발 가야: "다른 나? 재밌다! 만나보고 싶어!"
  - 공포 가야: "…나 말고 다른 나? …무서워."
- **모든 페르소나가 "다른 가야"를 모르지만, 반응에 본체의 성향이 공통으로 깔림**
- 추리 로그: 공통 금기, 공통 취향, 공통 감정 패턴이 자동 하이라이트

### S3. 깨달음 — 오후
- 추리 시스템에서 연결 완성:
  - "모든 가야가 같은 취향을 가짐"
  - "모든 가야가 같은 금기에 반응"
  - "모든 가야가 같은 사람을 무서워함"
  - "공통 키워드: '같이', '무서워', '이번에는'"
- 시스템 알림: **[추리 도달: 복수의 가야는 하나의 존재를 분해한 페르소나]**
- 테이 내면: "이 가야들은… 전부 같은 사람이야."

### S4. 머리색 인식 — 오후
- 이 시점에서 플레이어가 의식적으로 머리색을 확인
- 모든 페르소나 가야: 검은 머리
- 15화에서 봤던 은색 머리끝을 떠올림
- 추리 로그: **"검은 머리 = 조각 / 흰 머리 = 본체?"**
- 아직 흰 머리 가야는 만나지 못함

### S5. 거점 — 저녁
- 기본 가야와 대화
- 가야: "오늘 하루종일 뭐 했어?"
- 선택지:
  - "너를 이해하려고 했어."
  - "그냥 돌아다녔어."
- 1번: 가야 멈춤. 긴 침묵. "…다 이해하면… 어떡할 건데?"
- **"다 이해하면"이라는 표현 = 가야가 본체 구조를 알고 있다는 추가 암시**

### S6. 밤
- 밤 사냥 진행. 이번에는 몬스터 중 하나가 검은 머리를 가진 듯한 실루엣.
- 확실하지 않지만 불쾌한 기분.
- **화면 암전. EP.16 END.**
