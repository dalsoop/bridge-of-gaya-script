# EP.18 — 루프의 증거

> **루프 5회차 / Day 3 (핵심 화)**

## 개요
서고에서 과거 루프 기록 전체 발견. 기사단 NPC들의 대사가 전부 루프 인식이었음을 한꺼번에 깨달음.

## 등장인물
- 테이, 발터, 세라, 카이, 마그누스, 엘리시아

## 씬 구성

### S1. 서고 — 오전
- 14화에서 열린 구역보다 더 안쪽이 해금됨
- **과거 루프 기록 전체본** 발견
- 기록 내용:
  - 날짜가 동일한 5일이 수백 번 반복
  - 각 기록마다 "테이"가 다른 선택을 함
  - 어떤 기록: "가야를 죽였다. 루프 종료." (해방)
  - 어떤 기록: "죽이지 못했다. 반복." (유지)
  - 어떤 기록: "진실에 도달. 하지만 선택 불가. 루프 오류." (칼리스의 탄생)
  - 마지막 기록은 현재 루프에 해당. 아직 비어있음.

### S2. NPC 대사 총정리 — 점심
- 플레이어가 추리 로그에서 기사단 NPC들의 대사를 재확인:
  - 발터: "새로 왔나? …아닌가." → **루프 인식**
  - 세라: "이 길은 예전에도 걸었던 것 같은데?" → **루프 기억**
  - 카이: "패턴이 있다. 매번 같은 순서로 무너진다." → **루프 패턴 관찰**
  - 마그누스: "몬스터가 죽을 때 이상한 소리 안 들려?" → **가야 비명 인식**
  - 엘리시아: "이 상처… 바깥에서 생긴 게 아니야." → **내부 연결 인식**
- 시스템 알림: **[추리 도달: NPC들은 모두 루프를 부분적으로 인식하고 있었음]**
- **기사단은 기억 공간의 일부이지만, 가야의 기억 속 인물들이기에 루프의 잔여를 느끼고 있었다**

### S3. NPC 재방문 — 오후
- 각 NPC에게 "넌 루프를 알고 있었어?"라고 물으면:
  - 발터: "…루프? 뭔 소리야. …근데 가끔 꿈같은 기분이 들긴 해."
  - 카이: "…알았다면 벌써 멈췄겠지." (눈빛은 아는 듯)
  - 엘리시아: "…세상이 반복되는 건 아닐까 생각한 적은 있어요."
- **자각은 없지만 감각은 있음**

### S4. 거점 — 저녁
- 가야와 대화
- 테이: "서고에서 기록을 봤어. 이 세계는 반복되고 있어."
- 가야: "…"
- 가야: "…알아."
- 가야: "근데 어쩔 수 없잖아."
- **짧지만 강력한 대사. 가야는 이미 알고 있지만 어쩔 수 없음을 받아들인 상태.**
- 22화에서 더 깊이 파고듦.

### S5. 밤
- 밤 사냥. 일반 진행.
- **화면 암전. EP.18 END.**
