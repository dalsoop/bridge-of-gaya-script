# EP.06 — 반복의 냄새

> **루프 1→2 전환 / 다시 Day 1**

## 개요
5일이 지나고 다시 1일차. 루프가 발생. 가야는 잊은 듯하지만 미세한 데자뷔. 플레이어가 처음으로 "반복"을 의식.

## 등장인물
- 테이, 가야, 카이, 발터

## 씬 구성

### S1. 방 안 — 아침
- 테이가 눈을 뜸
- 1화와 같은 방. 같은 각도.
- 문 앞에 가야가 서 있음
- 가야: "…안녕."
- **1화와 똑같은 대사. 하지만 이번엔:**
- 가야: "…어? 왠지 이미 말했던 것 같은데."
- 가야: "…기분 탓이겠지."
- **데자뷔 반응. 1화의 "또 왔네"보다 약한 버전 — 루프마다 기억 유지량이 다르다는 암시**

### S2. 거점 — 아침
- 발터: "오, 새로운 놈인가? …잠깐, 어제도 본 것 같은데?"
- 발터: "…아니다. 내가 늙어서 헷갈리나 보다."
- **거점 상태도 리셋. 하지만 NPC들도 미세한 기억 잔재**

### S3. 장소 탐색 — 오전~점심
- 장소를 다시 가보면 가야가 1회차와 비슷하게 반응
- 하지만 일부 대사가 미묘하게 다름
  - 1회차: "여기 처음이네!"
  - 2회차: "여기… 처음인데 왜 익숙하지?"
- 플레이어만 차이를 알아챔 (가야는 의식 못 함)

### S4. 전망대 — 오후
- 카이를 다시 만남
- 카이: "패턴이 있다. 매번 같은 순서로 무너진다."
- 이번에는 추가 대사:
- 카이: "…너도 느꼈지? 이건 처음이 아니야."
- 선택지:
  - "무슨 뜻이야?"
  - "나도 그런 느낌이야."
  - (무시)
- 1번: 카이: "…말해줘도 믿지 않을 거다." 대화 끝.
- 2번: 카이가 잠깐 놀란 표정. "…역시." 하지만 더 말하지 않음.

### S5. 거점 — 저녁
- 가야: "오늘 왠지 하루가 되게 빨리 간 것 같아."
- 가야: "…아까도 이런 대화 했던 것 같은데."
- 테이가 리본에 대해 물으면:
  - 가야: "리본? …무슨 리본?" (3화의 선물을 모름)
  - **리본이 리셋됨. 하지만 7화에서 피 묻은 채로 돌아옴**
- 가야: "…내일도 같이 있을 거지?" (1화와 동일 대사)

### S6. 방 — 밤
- 밤 공포 반복
- 가야: (잠들기 전) "…왜 이렇게… 무서운 걸까. 처음인데."
- **"처음"이라고 말하지만, 몸은 이미 공포를 기억하고 있다**
- **화면 암전. "다시" DAY 1.**
- 화면 하단에 작게: **[Loop 2]**
- 이것이 플레이어에게 "루프"를 공식적으로 인지시키는 첫 순간

## 떡밥 정리

| 요소 | 유형 | 회수 |
|------|------|------|
| 데자뷔 반응 | Payoff (부분) | "또 왔네"(1화)의 약화판 |
| NPC 기억 잔재 | Plant | 18화: 전부 루프 인식 |
| 대사 미묘한 변화 | Plant | 17화: 기억 공간 세부 변화 |
| 리본 리셋 | Plant | 7화: 피 묻은 리본 귀환 |
| 카이 "처음이 아냐" | Plant | 18화: 루프 증거 |
| [Loop 2] 표시 | UI Plant | 21화: 루프 오류 |
| "처음인데" (공포) | Plant | 22화: 몸이 기억하는 공포 |

- NPC 대사 비교 시스템 (로그에서 확인 가능)
