# EP.27 — 분기점

> **루프 6회차(최종) / 마지막 아침 (핵심 화)**

## 개요
가야가 묻는다: "이번에는 어떻게 할 거야?" 플레이어의 선택과 이력에 따라 엔딩 루트가 갈린다.

## 등장인물
- 테이, 가야 (본체)

## 씬 구성

### S1. 거점 — 아침
- 마지막 아침.
- 가야가 평온한 얼굴로 앉아있음.
- 가야: "잘 잤어?"
- 가야: "…오늘로 끝이야."

### S2. 프롤로그의 거울 — 오전
- **[연출] 가야가 테이 앞에 선다. 프롤로그와 같은 구도. 같은 앵글. 같은 거리.**
- **하지만 프롤로그에서는 가야가 쓰러져 있었고, 지금은 서 있다.**
- **프롤로그에서는 테이만 선택했고, 지금은 가야도 선택할 수 있다.**

가야 (본체, 명확): "이번에는 어떻게 할 거야?"

가야 (본체): "선택해."

- **여기서 플레이어의 전체 이력(호감도, 이해도, 봉인석 복선 수집도, 권한 커맨드 여부)에 따라 분기.**

### S3. 분기 판정

**루트 A: 해방** (조건: 진실을 알고 + 가야를 죽이는 선택)
- 테이가 가야에게 다가감
- **[연출] 프롤로그와 완전히 같은 구도. 테이가 검을 듦. 하지만 이번에는 플레이어가 직접 선택한 것.**
- 가야: "…알았어. 이번에도 그거구나."
- 가야: "괜찮아."
- 가야 (미소): "이제… 안 무서워도 되네."
- **프롤로그의 "고마워"와 같은 미소. 같은 안도. 하지만 이번엔 플레이어가 왜인지 안다.**
- → **루트 A로 전환**

**루트 B: 유지** (조건: 진실을 알고 + 죽이지 않는 선택)
- 테이: "안 죽일 거야."
- 가야: "…그래?"
- 가야: "…고마워. …근데 나 계속 죽어."
- 가야: "…괜찮아?"
- → **루트 B로 전환**

**루트 C: 역전** (조건: 호감도 MAX + 스토리 진행 중심)
- 27화에서 가야가 먼저 움직임
- 가야: "…이번에는 내가 선택할게."
- 가야가 테이에게 다가감
- 가야가 미소
- 가야: "네가 계속 죄를 지으니까."
- 가야: "이번엔 내가 가져갈게."
- 가야가 테이를 찌름. 눈물. 웃음.
- → **루트 C로 전환**

**루트 D: 권한** (조건: 호감 충분 + 코드 수정 권한 커맨드 사용)
- 테이가 가야에게 권한을 줌
- 가야: "…정말?"
- 가야: "…고마워."
- 가야: "그러면… 시간을 멈출게."
- → **루트 D로 전환**

**루트 E: 에이전트** (조건: 봉인석 복선 전부 수집 + 봉인 장소 해금)
- 가야: "…하나 더 있어."
- 가야: "그 장소. 열 수 있어."
- 가야: "열면… 나는 여기서 나갈 수 있어."
- → **루트 E로 전환**

### S4. 분기 후
- 각 루트별 에피소드로 전환 (Phase 5 파일 참조)
- **화면 암전. EP.27 END. → 루트별 엔딩.**

## 떡밥 정리

| 요소 | 유형 | 회수 |
|------|------|------|
| "이번에는 어떻게 할 거야" | Final Question | 게임 전체 질문의 역전 |
| 칼리스 "이번에는 선택해" (20화) | Payoff | 분기 발생 |
| 모든 이전 선택 | Payoff | 분기 조건으로 누적 반영 |
| 프롤로그 구도 재현 | Final Payoff | L10: 0화의 장면을 플레이어가 "직접 선택"으로 재현 |

## 게임 메커닉
- 전체 이력 기반 분기 판정
- 5개 루트 중 하나로 전환
- 조건 미충족 루트는 잠금 (차회 플레이에서 해금 유도)
