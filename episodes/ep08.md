# EP.08 — 패턴의 발견

> **루프 2회차 / Day 4~5**

## 개요
플레이어가 능동적으로 여러 가야를 비교하기 시작. 공통 반복 문장 발견. 추리 시스템 본격 시작.

## 등장인물
- 테이, 가야 (복수 페르소나)

## 씬 구성

### S1. 탐색 — 오전
- 7화 충격 이후 플레이어가 의도적으로 여러 장소 방문
- 각 장소의 가야에게 동일한 질문을 던짐
- 질문: "넌 뭘 좋아해?" / "넌 뭘 무서워해?" / "나한테 바라는 게 뭐야?"
- **각 페르소나가 다르게 대답하지만, 특정 단어/문장만 겹침:**
  - 의존 가야: "같이 있고 싶어."
  - 냉소 가야: "…혼자가 편해. …그래도 같이 있으면… 나쁘지 않아."
  - 활발 가야: "같이 살았으면 좋았을 텐데!"
- **"같이"가 공통 키워드**

### S2. 비교 장면 — 점심
- 플레이어가 추리 로그를 확인
- 자동으로 공통 키워드가 하이라이트됨:
  - "같이" / "무서워" / "돌아와" / "이번에는"
- **추리 시스템**: 플레이어가 키워드를 조합해서 가설을 세울 수 있음
- 아직 결론은 안 남 — "왜 이렇게 겹치지?" 수준

### S3. 특정 장소 — 오후
- 한 장소에서 가야가 갑자기:
- 가야: "같이 살았으면 좋았을 텐데."
- 가야: "…어? 나 이 말 한 적 있었나?"
- 가야가 자기 대사에 데자뷔를 느낌
- **페르소나들 간에 기억이 약하게 공유되고 있다는 첫 힌트**

### S4. 거점 — 저녁
- 기본 페르소나 가야와 대화
- 가야: "오늘 많이 돌아다녔네. …나 찾아다닌 거야?"
- 선택지:
  - "응, 너를 더 알고 싶어서."
  - "그냥 돌아다닌 거야."
- 1번: 가야가 미소. 하지만 미묘하게 불안. "…다 알게 되면… 실망하지 않을까."
- 2번: 가야: "…그래." (약간 서운)

### S5. 밤
- 밤 사냥. 이번에는 몬스터 처치 중 리본은 안 드랍됨.
- 하지만 다른 아이템 드랍: **가야가 좋아한다고 말했던 음식의 포장지 같은 조각**
- 무의미해 보이지만 인벤토리에 저장됨
- **24화에서 "모든 드랍 = 가야의 잔해"로 회수**
- **화면 암전. EP.08 END.**
