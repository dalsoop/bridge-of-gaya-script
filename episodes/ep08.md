# EP.08 — 패턴의 발견

> **루프 2회차 / Day 4~5. 추리 시스템 본격 시작. 공통 키워드 발견.**

## 개요
7화 충격 후 플레이어가 능동적으로 여러 가야를 비교. 공통 반복 문장 발견. "같이"라는 키워드가 모든 페르소나에 관통.

## 등장인물
- 테이, 가야 (복수 페르소나)

## 씬 구성

### S1. 의도적 탐색 — 오전
```yaml
scene_id: ep08_s1
location: 복수 장소 순회
time: 오전
bgm: explore_01
tension: 4
pacing: setup → escalation
primary_emotion: determination
```

- 7화 충격 이후 플레이어가 의도적으로 모든 장소 방문
- 각 페르소나에게 동일 질문: "넌 뭘 좋아해?" / "넌 뭘 무서워해?" / "나한테 바라는 게 뭐야?"

가야(의존) (작게): "같이 있고 싶어."

가야(냉소) (시선 회피): "…혼자가 편해. …그래도 같이 있으면… 나쁘지 않아."

가야(활발) (밝게): "같이 살았으면 좋았을 텐데!"

- **"같이"가 공통 키워드. L6 Plant.**

### S2. 추리 시스템 — 점심
```yaml
scene_id: ep08_s2
location: 거점(추리 화면)
time: 점심
bgm: library_01
tension: 5
pacing: escalation
primary_emotion: discovery
```

- 플레이어가 추리 로그 확인
- 자동 하이라이트: **"같이" / "무서워" / "돌아와" / "이번에는"**
- 아직 결론 안 남 — "왜 이렇게 겹치지?" 수준

[시스템] 추리 로그 업데이트: 공통 키워드 발견.

### S3. 기억 공유 — 오후
```yaml
scene_id: ep08_s3
location: 탐색구역
time: 오후
bgm: explore_dark_01
tension: 5
pacing: escalation
primary_emotion: unease
```

- 한 장소에서 가야가 갑자기:

가야 (멈춤): "같이 살았으면 좋았을 텐데."

가야 (갸웃): "…어? 나 이 말 한 적 있었나?"

- **페르소나 간 기억이 약하게 공유. L6 강화. 16화 "하나의 본체" 연결.**

### S4. 거점 — 저녁
```yaml
scene_id: ep08_s4
location: 거점_방
time: 저녁
bgm: evening_01
tension: 4
pacing: setup
primary_emotion: warmth + doubt
romance: push
intimacy: 4→4.5
```

가야(기본) (조용): "오늘 많이 돌아다녔네. …나 찾아다닌 거야?"

- 선택지:
  1. "응, 너를 더 알고 싶어서." → gaya_comfort+2, flags: [honest_interest]
  2. "그냥 돌아다닌 거야." → flags: []

- 1번:

가야 (미소, 하지만 불안): "…다 알게 되면… 실망하지 않을까."

- **24화 Plant: "넌 이미 다 봤잖아." 가야는 진실이 드러날까 두려움.**

### S5. 밤
```yaml
scene_id: ep08_s5
location: 사냥터
time: 밤
bgm: battle_01
tension: 4
pacing: action
```

- 밤 사냥. 리본은 안 드랍됨.
- 다른 아이템 드랍: **가야가 좋아한다고 말했던 음식의 포장지 같은 조각**
- 무의미해 보이지만 인벤토리에 저장

[아이템] 음식 포장 조각 — 출처 불명

- **24화에서 "모든 드랍 = 가야의 잔해"로 회수. L3 확장.**

[연출] 화면 암전. "EP.08 END."
