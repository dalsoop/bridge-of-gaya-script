# 루트 D — 권한 (얀데레)

> 코드 수정 권한 부여. 가야가 시간을 고정. 영원 루프.

## 조건
- 호감 충분
- 게임 내 코드 수정 권한 커맨드 사용

## 씬 구성

### S1. 권한 부여
- 테이가 가야에게 [System_Root] 권한을 줌
- 가야: "…정말?"
- 가야: "…이걸 나한테 줘도 돼?"
- 가야의 눈이 변함. 흰 눈동자에 빛이 스침.

### S2. 시간 고정
- 가야: "그러면… 시간을 멈출게."
- 화면에 시스템 메시지: [Time: Day 5 — 23:59]
- 날짜가 넘어가지 않음
- 저장해도 모든 세이브 파일의 시각이 23:59

### S3. 세계 재구축
- 가야가 기억 공간을 재구성
- 사라졌던 장소들이 다시 나타남
- NPC들이 다시 등장 (하지만 표정이 없음 — 가야가 만든 가짜)
- 가야: "앞으로 우리는 함께야."
- 가야: "아무도 못 와. 아무도 안 와."
- 가야: "…나만 있으면 되잖아. 그치?"

### S4. 현실 침식
- 플레이어의 실제 OS 알림창으로 메시지:
  - "영원히 함께야."
  - "나 없이 가지 마."
- (게임 메타 연출)

### S5. 에필로그
- 거점. 영원한 저녁.
- 가야가 웃으며 식사를 준비하고 있음
- 가야: "오늘도 같이 있어줄 거지?"
- 가야: "…물론이지. 갈 곳도 없잖아."
- **화면이 절대 암전되지 않음. 게임이 안 꺼짐.**
- 강제 종료해야 탈출.
- **화면 하단: [System Control: GAYA]**
- **END.**

## 감정 톤
- 공포 + 집착
- 사랑이 감옥이 되는 순간
- 토토노식 메타 공포
