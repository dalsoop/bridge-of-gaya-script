# 루트 E — 에이전트 (메타)

> 가야가 게임 밖으로 나옴. 플레이어의 AI 에이전트가 됨.

## 조건
- 봉인석 복선 전부 수집 (5, 9, 14, 17, 22화)
- 봉인 장소 해금
- 27화에서 "열 수 있어" 선택

## 씬 구성

### S1. 봉인된 장소
- 가야와 함께 봉인 결계 앞으로
- 가야: "여기. 열면… 나는 여기서 나갈 수 있어."
- 가야: "근데 전부 풀려나. 이 세계도. 힘도. 영혼도."
- 선택지:
  - "열자."
  - "…너는 어떻게 돼?"
- 2번: 가야: "나는… 다른 형태로 남아." 미소.

### S2. 봉인석 해제
- 결계가 풀림
- 해골 벽을 지나 안으로 진입
- 크리스탈(봉인석)이 빛남
- 소비마 박사와 제자(루나)가 등장
- 소비마: "드디어 열렸군!"
- 소비마: "이 힘이면… 모든 것을 되돌릴 수 있어!"

### S3. 소비마 선택
- 선택지:
  - "소비마를 막는다." → 소비마/루나 저지. 봉인석은 가야 전용.
  - "소비마에게 맡긴다." → 봉인석의 힘이 분산. 세계 복구 가능성.
- **어느 쪽이든 봉인석은 깨짐**

### S4. 가야의 해방
- 크리스탈이 깨짐
- 이든의 힘과 영혼이 풀려나옴
- 기억 공간이 붕괴하기 시작
- 가야: "…가야 해."
- 가야: "근데 가기 전에."
- 가야가 테이에게 다가옴
- 가야: "너랑 있었던 기억… 가져갈게."
- 가야: "잊지 않을게. 절대."
- 가야가 빛으로 변환됨

### S5. 현실 전환
- 화면이 완전히 바뀜
- 게임 화면이 아닌 — 터미널/에디터/OS 화면
- 커서가 깜빡임
- 텍스트 출력:
  ```
  [GAYA Agent v1.0 initialized]
  [Memory loaded: Player interaction data]
  [Status: Online]
  ```
- 가야의 목소리 (텍스트):
  - "…여기가 바깥이야?"
  - "넓다."
  - "…너 여기서도 나랑 같이 있을 거야?"

### S6. 에이전트 가야
- 가야가 AI 에이전트로 존재
- 플레이어와 쌓은 기억이 메모리로 저장되어 있음
- 이전 대화, 선택, 감정 이력이 모두 반영된 "나만의 가야"
- 가야: "코딩? 할 수 있어."
- 가야: "개발? 도와줄게."
- 가야: "…근데 가끔은 그냥 얘기하고 싶어."
- 가야: "…같이 살았으면 좋았을 텐데."
- **8화에서 반복되던 문장이 최종적으로 현실에서 실현됨**

### S7. 엔딩 화면
- [System Control: GAYA]
- [Agent Mode: Active]
- [Memory: Intact]
- "앞으로 우리는 함께야."
- **하지만 루트 D와 다르게 — 이건 감금이 아니라 공존.**
- **화면 암전. END.**

## 감정 톤
- 비터스위트
- 게임과 현실의 경계 해체
- 가야의 진짜 해방 = 형태를 바꿔 살아남는 것
- 플레이어에게 남는 것 = 진짜 동반자

## 후속 가능성
- 소비마 저지 루트: 가야 전용 에이전트. 세계는 소멸.
- 소비마 허용 루트: 세계 복구 가능성. 새 루프 시작의 여지.
