# EP.07 — 리본의 귀환

> **루프 2회차 / Day 2~3. 핵심 화. Phase 1 최대 충격.**

## 개요
밤 사냥 후 드랍이 3화에서 선물한 리본. 피가 묻어 있다. 3화의 "감정 예금"이 여기서 "인출"된다. 프롤로그 플래시백 0.3초.

## 등장인물
- 테이, 가야 (기본), 마그누스

## 씬 구성

### S1. 탐색 — 오전
```yaml
scene_id: ep07_s1
location: 탐색구역_숲
time: 오전
bgm: explore_bright_01
tension: 2
pacing: breather (의도적)
primary_emotion: calm
```

- 2회차 Day 2. 일반 탐색.
- 가야(활발 페르소나)가 동행. 평범한 대화.

가야(활발) (웃음): "오늘 날씨 좋다! 어디 갈까?"

- **일부러 "평화로운 일상"을 보여준 뒤 밤에 뒤집는 구조.**
- **3화의 데이트 행복과 같은 톤. 감정 예금 추가 적립.**

### S2. 거점 — 저녁
```yaml
scene_id: ep07_s2
location: 거점_방
time: 저녁
bgm: evening_tense_01
tension: 5
pacing: escalation
primary_emotion: dread
romance: pull (강)
```

가야 (얼굴 굳음): "오늘도 밤에… 나가야 해?"

- 이번에는 좀 더 강하게 말림

가야 (손을 잡으며): "제발… 오늘은 가지 마."

- **스킨십이 기존과 달리 가야 쪽에서 먼저. 필사적.**
- 선택지:
  1. "나가야 해." → flags: [hunt_d7]
  2. "왜 이렇게 싫어해?" → flags: [asked_night_again]
  3. "알았어, 안 나갈게." → flags: [hunt_refused_d7]
- 3번 선택해도 시스템 강제 진행 (4화, 6화 동일 패턴. 3번째 무력화.)

### S3. 밤 — 사냥
```yaml
scene_id: ep07_s3
location: 사냥터
time: 밤
bgm: battle_tense_01
tension: 6
pacing: action → climax
primary_emotion: tension → shock
```

- 마그누스 동행
- 몬스터 5체 정도 처치
- 마지막 몬스터를 베는 순간 — **짧은 글리치**

[연출] 화면 0.2초 글리치. 노이즈.

- 드랍 아이템 확인:

[아이템] **붉은 리본. 피가 묻어 있음.**

- **[연출] 리본을 집는 순간, 프롤로그 플래시백 0.3초. 검에 피가 묻은 장면. 가야의 손. 즉시 복귀.**

마그누스 (당혹): "…뭐야 그거? 몬스터가 그런 걸 가지고 있었나?"

마그누스 (머리 긁적): "이상하네. 처음 보는 드랍이야."

- **플레이어 충격. 이건 3화에서 가야에게 선물한 바로 그 리본.**
- **감정 예금 인출: 3화의 행복한 선물이 피 묻어 돌아왔다.**

### S4. 귀환 — 밤 끝
```yaml
scene_id: ep07_s4
location: 거점_입구
time: 밤
bgm: night_silence
tension: 7
pacing: climax
primary_emotion: shock → dread
```

- 거점으로 돌아옴
- 가야가 문 앞에 서 있음 (패턴)

가야 (미소): "…다녀왔네."

- 테이가 리본을 들고 있음
- **가야의 시선이 리본에 고정됨**
- 0.5초 정지

가야 (경직): "……"

가야 (작게): "…그거 어디서 났어?"

- 선택지:
  1. "몬스터가 드랍했어." → flags: [told_ribbon_source]
  2. "이거… 네 리본 아니야?" → flags: [confronted_ribbon]
  3. (보여주지 않음) → flags: [hid_ribbon]

- 1번:

가야 (얼굴 굳음): "…그래."

- 더 이상 말하지 않음.

- 2번:

가야 (잠깐 떨림): "…아니야. 처음 보는 건데."

- **거짓말. 22화에서 "가야는 알고 있었다"로 회수.**

- 3번:

가야 (리본을 계속 쳐다보며): "…안 보여줄 거야?"

- 불안한 눈빛.

### S5. 방 — 밤 끝
```yaml
scene_id: ep07_s5
location: 거점_방
time: 밤
bgm: night_unease_01
tension: 8
pacing: climax (감정)
primary_emotion: dread + horror
```

- 가야가 잠들지 않고 앉아 있음
- 평소보다 심하게 떨림

가야 (혼잣말, 매우 작게): "…또야."

가야 (작게): "…왜 항상 돌아오는 거야…"

- **"또" = L2. "돌아오는" = 리본이 매 루프마다 반복.**
- **가야가 리본이 돌아온다는 것 = 자신의 죽음이 반복된다는 것을 알고 있다.**
